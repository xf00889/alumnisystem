# User Management

## Overview

The User Management feature allows administrators to view, search, filter, and manage all user accounts in the NORSU Alumni System. This includes viewing user details, editing user information, managing user roles and permissions, and activating or deactivating accounts.

## Who Can Use This Feature

- **Admin Users**: Full access to all user management features
- **Staff Users**: Limited access based on assigned permissions

## Prerequisites

- Admin or staff account with appropriate permissions
- Access to the Django admin panel

## Key Features

- View comprehensive list of all users
- Search users by username, email, name
- Filter users by staff status, active status, and groups
- View detailed user information including profile data
- Edit user information and permissions
- Manage user roles and group memberships
- Activate or deactivate user accounts
- View user activity and login history
