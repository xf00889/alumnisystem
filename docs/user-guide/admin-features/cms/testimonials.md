# Testimonials

## Overview

Testimonials Management allows you to create and display success stories, reviews, and endorsements from alumni on the homepage. Testimonials provide social proof and help build trust with potential members by showcasing real experiences and achievements.

## Who Can Use This Feature

- Admin users with staff privileges
- Users with CMS management permissions
- Marketing staff
- Alumni relations staff

## Prerequisites

- Admin account with appropriate permissions
- Logged into the system
- Access to CMS Dashboard
- Testimonials from alumni (with permission to publish)
- Photos of alumni (optional but recommended)

## How to Access

### Method 1: Through CMS Dashboard
1. Navigate to CMS Dashboard (`/cms/dashboard/`)
2. Click on "Testimonials" or "Manage Testimonials"
3. You will see the list of all testimonial entries

### Method 2: Direct URL
- Navigate to `/cms/testimonials/`

## Key Features

### 1. View All Testimonials
- List view of all testimonial entries
- Sort by order or name
- Filter by active status
- Pagination for large lists
- Photo preview in list

### 2. Add New Testimonials
- Create new testimonial entries
- Upload photos
- Add quotes and attribution
- Set display order
- Activate or deactivate

### 3. Edit Testimonials
- Update testimonial information
- Change photos
- Modify quotes
- Update attribution details
- Adjust display order

### 4. Delete Testimonials
- Remove outdated testimonials
- Confirmation required
- Permanent action

### 5. Testimonial Ordering
- Control display sequence
- Lower numbers appear first
- Rotate testimonials regularly

## Step-by-Step Guide

### Task 1: View Testimonials List

1. Access Testimonials from CMS Dashboard
2. View the list of all testimonial entries
3. Observe the following information:
   - Person's name
   - Position/title
   - Company/organization
   - Display order
   - Active status
   - Photo preview
4. Use pagination to navigate through multiple pages
5. Click on names to view full details

**Expected Result**: Complete list of all testimonials with preview information and photos.

### Task 2: Add a New Testimonial

1. From the Testimonials list, click "Add Testimonial" or "Create New"
2. Fill in the testimonial details:

   **Name** (required):
   - Enter full name of the person
   - Maximum 200 characters
   - Use proper capitalization
   - Example: "Maria Santos"
   
   **Position** (required):
   - Enter job title or role
   - Maximum 200 characters
   - Be specific and professional
   - Example: "Senior Software Engineer"
   
   **Company** (optional):
   - Enter company or organization name
   - Maximum 200 characters
   - Include if relevant and adds credibility
   - Example: "Google Philippines"
   
   **Quote** (required):
   - Enter the testimonial text
   - Keep it authentic and specific
   - 2-4 sentences recommended
   - Focus on specific benefits or outcomes
   - Use first-person perspective
   - Include concrete details when possible
   - Example: "The NORSU Alumni Network helped me connect with industry leaders and land my dream job. The mentorship program was invaluable in my career transition. I'm grateful for this amazing community."
   
   **Image** (optional but recommended):
   - Click "Choose File" to upload photo
   - Recommended size: 300x300 pixels or larger
   - Use professional headshot
   - Supported formats: JPG, PNG
   - Square aspect ratio works best
   - Get permission before using photo
   
   **Order**:
   - Enter a number for display order
   - Lower numbers appear first
   - Default: 0
   - Use increments of 10 (10, 20, 30...)
   - Consider rotating testimonials regularly
   
   **Is Active**:
   - Check to make testimonial visible
   - Uncheck to hide without deleting

3. Review all entered information
4. Verify you have permission to publish
5. Click "Save" or "Create Testimonial"

**Expected Result**: New testimonial is created and appears on the homepage testimonials section.

### Task 3: Edit an Existing Testimonial

1. From the Testimonials list, find the testimonial to edit
2. Click the "Edit" button or person's name
3. Update any of the following fields:
   - Name
   - Position
   - Company
   - Quote
   - Image (upload new or clear existing)
   - Order
   - Active status
4. To change the photo:
   - Upload a new photo to replace current one
   - Check "Clear" to remove photo without replacement
5. Click "Save" or "Update Testimonial"

**Expected Result**: Testimonial is updated with new information. Changes appear on the homepage.

### Task 4: Collect and Add Testimonials

**Process for Gathering Testimonials**:

1. **Identify Candidates**:
   - Successful alumni
   - Active community members
   - Recent job placement success stories
   - Mentorship program participants
   - Event attendees with positive feedback

2. **Request Testimonial**:
   - Send personalized email or message
   - Explain how testimonial will be used
   - Request permission to use name and photo
   - Provide guidance on what to include
   - Suggest 2-4 sentences

3. **Review Submission**:
   - Check for authenticity
   - Verify specific details
   - Ensure appropriate language
   - Confirm permission to publish

4. **Add to System**:
   - Create new testimonial entry
   - Enter all details accurately
   - Upload photo (with permission)
   - Set appropriate order
   - Activate testimonial

5. **Follow Up**:
   - Thank the person
   - Share link to published testimonial
   - Keep record of permission

**Expected Result**: Authentic, compelling testimonials from real alumni.

### Task 5: Organize Testimonials for Impact

**Organization Strategies**:

**By Diversity (Recommended)**:
- Mix of different industries
- Various graduation years
- Different career stages
- Geographic diversity
- Gender balance

**By Theme**:
- Career success: Order 10-19
- Networking benefits: Order 20-29
- Mentorship impact: Order 30-39
- Community value: Order 40-49

**By Recency**:
- Recent testimonials first
- Rotate older testimonials out
- Keep content fresh

**Steps to Organize**:
1. Review all testimonials
2. Identify diverse representation
3. Assign order values strategically
4. Edit each testimonial and update order
5. Save all changes
6. Review homepage display

**Expected Result**: Testimonials showcase diverse success stories and appeal to various audiences.

### Task 6: Rotate Testimonials

**Why Rotate**:
- Keep content fresh
- Showcase different success stories
- Maintain user interest
- Highlight various benefits

**Rotation Strategy**:

1. **Monthly Rotation**:
   - Deactivate current testimonials
   - Activate new set of testimonials
   - Keep 3-6 active at a time

2. **Seasonal Themes**:
   - Career-focused during job season
   - Event-focused before major events
   - Mentorship-focused during recruitment

3. **Steps to Rotate**:
   - Identify testimonials to deactivate
   - Edit and uncheck "Is Active"
   - Identify testimonials to activate
   - Edit and check "Is Active"
   - Save all changes
   - Review homepage

**Expected Result**: Fresh, relevant testimonials that maintain user engagement.

### Task 7: Deactivate a Testimonial

1. From the Testimonials list, find the testimonial
2. Click "Edit" on the testimonial
3. Uncheck the "Is Active" checkbox
4. Click "Save"

**Use Cases**:
- Rotating testimonials
- Person requests removal
- Outdated information
- Testing before making public

**Expected Result**: Testimonial no longer appears on the homepage but remains in admin list.

### Task 8: Delete a Testimonial

1. From the Testimonials list, find the testimonial to delete
2. Click the "Delete" button
3. Review the confirmation page:
   - Testimonial details are displayed
   - Warning about permanent deletion
4. Confirm you want to delete
5. Click "Yes, delete" or "Confirm deletion"

**Use Cases**:
- Person requests permanent removal
- Duplicate entries
- Inappropriate content
- Fake testimonials

**Expected Result**: Testimonial is permanently removed from the system.

## Testimonial Content Best Practices

### Authentic Quotes
- Use real testimonials from real people
- Don't fabricate or heavily edit quotes
- Keep the person's voice and style
- Minor edits for clarity are acceptable
- Always get permission before publishing

### Effective Testimonials Include:
- **Specific Benefits**: "helped me land my dream job" not "was helpful"
- **Concrete Results**: "connected with 50+ alumni" not "made connections"
- **Personal Story**: Brief context about their situation
- **Emotional Impact**: How it made them feel
- **Call to Action**: Implicit encouragement for others

### Quote Length
- 2-4 sentences ideal
- 50-100 words recommended
- Long enough to be meaningful
- Short enough to be scannable
- Focus on one main benefit or story

### Attribution
- Full name (with permission)
- Current position and company
- Graduation year (optional)
- Location (optional)
- Adds credibility and relatability

### Photos
- Professional headshots work best
- Good lighting and clear face
- Neutral or professional background
- Square aspect ratio (1:1)
- High resolution (at least 300x300)
- Always get permission to use

## Tips and Best Practices

1. **Permission**: Always get written permission before publishing
2. **Authenticity**: Use real testimonials; never fabricate
3. **Diversity**: Showcase variety in industries, roles, and backgrounds
4. **Specificity**: Encourage specific details and outcomes
5. **Recency**: Keep testimonials current and relevant
6. **Rotation**: Regularly update displayed testimonials
7. **Quality Over Quantity**: Better to have 3 great testimonials than 10 mediocre ones
8. **Visual Appeal**: Include photos whenever possible
9. **Verification**: Verify testimonials are from real alumni
10. **Follow-Up**: Thank contributors and share published testimonials

## Legal and Ethical Considerations

### Permission and Consent
- Get explicit permission to publish
- Specify where testimonial will appear
- Request permission for photo use
- Keep records of permissions
- Honor requests for removal

### Privacy
- Don't publish without permission
- Respect privacy preferences
- Be careful with sensitive information
- Consider using first name only if requested

### Authenticity
- Never fabricate testimonials
- Don't heavily edit to change meaning
- Disclose if incentives were provided
- Maintain integrity and trust

### Updates
- Update if person's information changes
- Remove if person requests
- Keep contact information current
- Respect alumni preferences

## Important Notes

- **Photo Storage**: Images are stored in `media/cms/testimonials/` directory
- **Permission Records**: Keep records of permissions separately
- **Order Conflicts**: Multiple testimonials can have the same order value
- **Deletion**: Deleting a testimonial is permanent and cannot be undone
- **Display Limit**: Homepage may limit number of displayed testimonials (typically 3-6)

## Troubleshooting

### Testimonial Not Appearing on Homepage

**Issue**: Created or edited testimonial doesn't show on the website

**Solutions**:
- Verify "Is Active" checkbox is checked
- Clear browser cache and refresh
- Check display order
- Verify homepage template includes testimonials section
- Check if there's a display limit (e.g., only first 6 shown)
- Wait a few moments for cache to clear

### Photo Not Displaying

**Issue**: Uploaded photo doesn't appear in testimonial

**Solutions**:
- Verify image file format (JPG or PNG)
- Check image file size (should be under 5MB)
- Ensure image uploaded successfully
- Verify media folder permissions
- Try uploading a different image
- Use square aspect ratio image
- Check if image is corrupted

### Photo Quality Issues

**Issue**: Photo appears pixelated or distorted

**Solutions**:
- Upload higher resolution image (at least 300x300)
- Use square aspect ratio to avoid distortion
- Check original image quality
- Avoid excessive compression
- Use PNG for better quality
- Ensure proper image dimensions

### Quote Too Long

**Issue**: Long quote breaks layout or is hard to read

**Solutions**:
- Edit quote to be more concise
- Focus on main point or benefit
- Aim for 2-4 sentences
- Consider splitting into multiple testimonials
- Check mobile view for layout issues

### Testimonials Out of Order

**Issue**: Testimonials don't display in expected sequence

**Solutions**:
- Review order values for all testimonials
- Ensure logical ordering
- Use consistent increments (10, 20, 30)
- Check if multiple testimonials have same order value
- Verify template respects order field

### Cannot Save Testimonial

**Issue**: Save button doesn't work or shows errors

**Solutions**:
- Check for validation errors (red text near fields)
- Ensure required fields are filled (Name, Position, Quote)
- Verify character limits on fields
- Try refreshing the page and re-entering data
- Check image file size if uploading
- Contact system administrator if issue persists
