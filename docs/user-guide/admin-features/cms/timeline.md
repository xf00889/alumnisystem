# Timeline

## Overview

Timeline Management allows you to create and maintain a chronological history of important events, milestones, and achievements for the university and alumni network. The timeline is displayed on the About page to showcase the institution's journey and growth.

## Who Can Use This Feature

- Admin users with staff privileges
- Users with CMS management permissions
- Content managers

## Prerequisites

- Admin account with appropriate permissions
- Logged into the system
- Access to CMS Dashboard
- Historical information about events and milestones

## How to Access

### Method 1: Through CMS Dashboard
1. Navigate to CMS Dashboard (`/cms/dashboard/`)
2. Click on "Timeline Items" or "Manage Timeline"
3. You will see the list of all timeline items

### Method 2: Direct URL
- Navigate to `/cms/timeline-items/`

## Key Features

### 1. View All Timeline Items
- List view of all timeline entries
- Sort by year or order
- Filter by active status
- Pagination for large lists

### 2. Add New Timeline Items
- Create new historical entries
- Set year or date
- Add descriptions
- Choose icons
- Set display order

### 3. Edit Timeline Items
- Update event information
- Change years or dates
- Modify descriptions
- Update icons
- Adjust display order

### 4. Delete Timeline Items
- Remove outdated or incorrect entries
- Confirmation required
- Permanent action

### 5. Timeline Ordering
- Control chronological display
- Lower order numbers appear first
- Organize by importance or date

## Step-by-Step Guide

### Task 1: View Timeline Items List

1. Access Timeline Items from CMS Dashboard
2. View the list of all timeline entries
3. Observe the following information:
   - Year or date
   - Event title
   - Display order
   - Active status
   - Creation date
4. Use pagination to navigate through multiple pages
5. Click on titles to view full details

**Expected Result**: Complete list of all timeline items in chronological or ordered sequence.

### Task 2: Create a New Timeline Item

1. From the Timeline Items list, click "Add Timeline Item" or "Create New"
2. Fill in the timeline item details:

   **Year** (required):
   - Enter year or date of the event
   - Maximum 10 characters
   - Examples: "2004", "May 2010", "2015-2016"
   - Can be a single year or range
   
   **Title** (required):
   - Enter event title or milestone name
   - Maximum 200 characters
   - Be concise and descriptive
   - Example: "University Established"
   
   **Description** (required):
   - Enter detailed description of the event
   - Explain significance and impact
   - Include relevant details
   - 2-4 sentences recommended
   - Example: "Negros Oriental State University was officially established through the merger of several educational institutions, marking a new era in higher education in the region."
   
   **Icon** (optional):
   - Enter Font Awesome icon class
   - Default: "fas fa-circle"
   - Examples:
     - "fas fa-graduation-cap" (education)
     - "fas fa-trophy" (achievement)
     - "fas fa-building" (establishment)
     - "fas fa-users" (community)
     - "fas fa-star" (milestone)
   - See Font Awesome documentation for more icons
   
   **Order**:
   - Enter a number for display order
   - Lower numbers appear first
   - Default: 0
   - Use chronological order or importance
   - Use increments of 10 (10, 20, 30...)
   
   **Is Active**:
   - Check to make item visible
   - Uncheck to hide without deleting

3. Review all entered information
4. Click "Save" or "Create Timeline Item"

**Expected Result**: New timeline item is created and appears in the list and on the About page timeline.

### Task 3: Edit a Timeline Item

1. From the Timeline Items list, find the item to edit
2. Click the "Edit" button or item title
3. Update any of the following fields:
   - Year
   - Title
   - Description
   - Icon
   - Order
   - Active status
4. Make necessary changes
5. Click "Save" or "Update Timeline Item"

**Expected Result**: Timeline item is updated with new information. Changes appear on the About page.

### Task 4: Organize Timeline Chronologically

1. View the Timeline Items list
2. Identify the chronological order of events
3. Assign order values based on timeline:
   - Earliest events get lowest numbers
   - Most recent events get highest numbers
   - Use increments of 10 for flexibility
4. Edit each timeline item and update the "Order" field
5. Save each item after updating order
6. Return to list view to verify chronological order

**Example Ordering**:
- 1950s events: Order 10, 20, 30
- 1960s events: Order 40, 50, 60
- 1970s events: Order 70, 80, 90
- And so on...

**Expected Result**: Timeline displays events in proper chronological sequence on the About page.

### Task 5: Update Timeline Icons

1. From the Timeline Items list, find items needing icon updates
2. Click "Edit" on each item
3. Update the "Icon" field with appropriate Font Awesome class
4. Choose icons that represent the event type:
   - Establishment: "fas fa-building"
   - Academic achievement: "fas fa-graduation-cap"
   - Award or recognition: "fas fa-trophy"
   - Expansion: "fas fa-expand-arrows-alt"
   - Partnership: "fas fa-handshake"
   - Innovation: "fas fa-lightbulb"
   - Community: "fas fa-users"
   - Research: "fas fa-flask"
5. Click "Save"

**Expected Result**: Timeline displays with appropriate visual icons for each event.

### Task 6: Deactivate a Timeline Item

1. From the Timeline Items list, find the item
2. Click "Edit" on the timeline item
3. Uncheck the "Is Active" checkbox
4. Click "Save"

**Use Cases**:
- Temporarily remove disputed information
- Hide items pending verification
- Test timeline without certain events

**Expected Result**: Timeline item no longer appears on the About page but remains in admin list.

### Task 7: Delete a Timeline Item

1. From the Timeline Items list, find the item to delete
2. Click the "Delete" button
3. Review the confirmation page:
   - Timeline item details are displayed
   - Warning about permanent deletion
4. Confirm you want to delete
5. Click "Yes, delete" or "Confirm deletion"

**Use Cases**:
- Remove incorrect information
- Delete duplicate entries
- Clean up outdated items

**Expected Result**: Timeline item is permanently removed from the system.

## Timeline Content Best Practices

### Year Format
- Use consistent format across all entries
- Single year: "2004"
- Year range: "2004-2005"
- Month and year: "June 2010"
- Decade: "1990s"
- Be consistent with your chosen format

### Event Titles
- Keep concise (3-6 words)
- Use active voice when possible
- Capitalize properly
- Focus on the main achievement
- Examples:
  - "University Established"
  - "First Graduating Class"
  - "New Campus Opened"
  - "International Accreditation Achieved"

### Event Descriptions
- Provide context and significance
- Include specific details (numbers, names, locations)
- Explain impact on university or alumni
- Keep to 2-4 sentences
- Use professional, formal tone
- Proofread for accuracy

### Icon Selection
- Choose icons that clearly represent the event
- Be consistent with icon style (all solid or all regular)
- Use recognizable symbols
- Don't overuse the same icon
- Test icon appearance on the timeline

### Chronological Organization
- Order events from oldest to newest (or vice versa)
- Group events by decade if many entries
- Highlight major milestones
- Include both achievements and challenges
- Balance quantity (not too many or too few)

## Tips and Best Practices

1. **Historical Accuracy**: Verify dates and facts before publishing
2. **Balanced Coverage**: Include events from all eras of the institution
3. **Regular Updates**: Add new milestones as they occur
4. **Visual Appeal**: Use varied icons to make timeline interesting
5. **Concise Writing**: Keep descriptions brief but informative
6. **Significance**: Focus on truly important events and milestones
7. **Consistency**: Maintain consistent formatting across all entries
8. **Proofreading**: Check for spelling and grammatical errors
9. **Sources**: Keep records of sources for historical information
10. **Review**: Periodically review timeline for accuracy and relevance

## Important Notes

- **Chronological Display**: Timeline typically displays in chronological order on the website
- **Order Field**: Use order field to control exact sequence if not purely chronological
- **Icon Library**: Icons use Font Awesome; ensure your theme includes Font Awesome
- **Deletion**: Deleting a timeline item is permanent and cannot be undone
- **Historical Records**: Consider keeping backup documentation of historical events

## Troubleshooting

### Timeline Item Not Appearing

**Issue**: Created or edited timeline item doesn't show on About page

**Solutions**:
- Verify "Is Active" checkbox is checked
- Clear browser cache and refresh
- Check display order
- Verify About page template includes timeline section
- Wait a few moments for cache to clear
- Check if there's a limit on displayed items

### Icons Not Displaying

**Issue**: Icons don't appear or show as boxes

**Solutions**:
- Verify Font Awesome is loaded on the page
- Check icon class name is correct (include "fas" or "far")
- Ensure icon exists in Font Awesome library
- Try a different, common icon (like "fas fa-circle")
- Check browser console for errors
- Verify Font Awesome version compatibility

### Timeline Out of Order

**Issue**: Events don't display in chronological sequence

**Solutions**:
- Review order values for all items
- Ensure chronological events have sequential order numbers
- Check if template has custom sorting logic
- Use consistent increments (10, 20, 30)
- Verify no duplicate order values causing conflicts

### Year Format Inconsistent

**Issue**: Years display in different formats

**Solutions**:
- Standardize year format across all entries
- Edit each item to use consistent format
- Choose one format and stick to it
- Update documentation for future entries

### Description Too Long

**Issue**: Long descriptions break timeline layout

**Solutions**:
- Edit description to be more concise
- Aim for 2-4 sentences maximum
- Focus on key information only
- Consider splitting into multiple timeline items
- Check mobile view for layout issues

### Cannot Save Timeline Item

**Issue**: Save button doesn't work or shows errors

**Solutions**:
- Check for validation errors (red text near fields)
- Ensure required fields are filled (Year, Title, Description)
- Verify icon class format is correct
- Try refreshing the page and re-entering data
- Check character limits on fields
- Contact system administrator if issue persists
