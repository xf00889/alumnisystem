# Donations

## Overview

The donation management system allows administrators to create and manage fundraising campaigns, process donations, verify payment proofs, and configure payment methods. This comprehensive system supports GCash payments and provides tools for campaign tracking, donor management, and financial reporting.

## Key Features

- **Campaign Management**: Create, edit, and manage fundraising campaigns
- **Donation Processing**: Review and verify donation payments
- **GCash Configuration**: Set up and manage GCash payment accounts
- **Payment Verification**: Verify payment proofs and update donation status
- **Donor Management**: Track donors and send receipts
- **Analytics**: View campaign performance and donation statistics

## Who Can Use These Features

- **User Role**: Admin users and staff members
- **Permissions Required**: Admin or staff privileges
- **Prerequisites**: Active admin account with appropriate permissions

## Common Tasks

1. **Create a New Campaign**: Set up a fundraising campaign with goals and settings
2. **Verify Donations**: Review payment proofs and approve donations
3. **Configure GCash**: Set up GCash account for receiving donations
4. **Send Receipts**: Send confirmation emails to donors
5. **View Analytics**: Track campaign performance and donation statistics

## Important Notes

- All donations require payment proof verification before being marked as completed
- GCash configuration must be set up before campaigns can accept donations
- Campaign visibility can be set to public or registered alumni only
- Donation verification should be completed within 24 hours of submission
