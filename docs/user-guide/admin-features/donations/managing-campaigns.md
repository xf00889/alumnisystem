# Managing Campaigns

## Overview

The campaign management feature allows administrators to view, edit, update status, and delete fundraising campaigns. You can monitor campaign progress, update campaign details, change visibility settings, and manage the lifecycle of campaigns from draft to completion.

## Who Can Use This Feature

- **User Role**: Admin users and staff members only
- **Permissions Required**: Admin or staff privileges
- **Prerequisites**: 
  - Active admin or staff account
  - Access to campaign management area
  - Existing campaigns to manage

## How to Access

1. Log in to your admin account
2. Navigate to the **Donations** section from the admin menu
3. Click on **Manage Campaigns**

## Key Features

- View all campaigns with filtering and search
- Edit campaign details and settings
- Update campaign status (Draft, Active, Paused, Completed, Cancelled)
- Delete campaigns (with restrictions)
- Monitor campaign progress and statistics
- Filter campaigns by status, type, and search terms
- View donation counts and amounts for each campaign

## Step-by-Step Guide

### Task 1: Viewing and Filtering Campaigns

This task covers accessing the campaign list and using filters to find specific campaigns.

1. **Access the campaign management page**
   - Navigate to **Donations > Manage Campaigns**
   - **Expected Result**: List of all campaigns appears with statistics

2. **View campaign statistics**
   - At the top of the page, view summary statistics:
     - Total Campaigns
     - Active Campaigns
     - Completed Campaigns
     - Draft/Paused Campaigns
   - **Expected Result**: Overview of campaign status distribution

3. **Use the search function**
   - Enter keywords in the search box
   - Search looks in campaign names, descriptions, and short descriptions
   - Press Enter or click Search
   - **Expected Result**: Filtered list shows matching campaigns

4. **Filter by campaign status**
   - Use the **Status** dropdown to filter:
     - All Statuses
     - Draft
     - Active
     - Paused
     - Completed
     - Cancelled
   - **Expected Result**: List shows only campaigns with selected status

5. **Filter by campaign type**
   - Use the **Campaign Type** dropdown
   - Select specific type (Scholarship, Infrastructure, etc.)
   - **Expected Result**: List shows only campaigns of selected type

6. **Clear filters**
   - Click **Clear Filters** or select "All" options
   - **Expected Result**: Full campaign list is restored

7. **View campaign details in list**
   - Each campaign card shows:
     - Campaign name and type
     - Goal amount and current amount raised
     - Progress bar and percentage
     - Number of donors
     - Status badge
     - Featured badge (if applicable)
     - Action buttons (Edit, View, Delete)
   - **Expected Result**: Quick overview of each campaign

### Task 2: Editing Campaign Details

This task covers updating existing campaign information and settings.

1. **Select campaign to edit**
   - From the campaign list, click **Edit** button on the desired campaign
   - Or click the campaign name to view details, then click **Edit Campaign**
   - **Expected Result**: Campaign edit form appears with current data

2. **Update campaign information**
   - Modify any of the following fields:
     - Campaign Name
     - Campaign Type
     - Short Description
     - Full Description
     - Featured Image
     - Goal Amount
     - Start Date
     - End Date
   - **Expected Result**: Changes are reflected in the form

3. **Update visibility settings**
   - Change **Visibility** between:
     - Registered Alumni
     - Public
   - Consider impact on current donors and campaign reach
   - **Expected Result**: Visibility setting is updated

4. **Modify donation settings**
   - Check/uncheck **Allow Donations** to enable/disable donations
   - Change **GCash Configuration** if needed
   - Note: Changing GCash account affects future donations only
   - **Expected Result**: Donation settings are updated

5. **Update featured status**
   - Check/uncheck **Is Featured** to change prominence
   - Remember to limit featured campaigns to 2-3 at a time
   - **Expected Result**: Featured status is updated

6. **Save changes**
   - Click **Save** or **Update Campaign**
   - **Expected Result**: Changes are saved and you're redirected to campaign detail page
   - Success message confirms update

7. **Verify changes**
   - Review the campaign detail page to confirm updates
   - Check the public-facing campaign page to see changes
   - **Expected Result**: All changes are visible and correct

### Task 3: Managing Campaign Status

This task covers changing campaign status throughout its lifecycle.

1. **Understand status options**
   - **Draft**: Campaign is being prepared, not visible to users
   - **Active**: Campaign is live and accepting donations
   - **Paused**: Campaign is temporarily suspended, visible but not accepting donations
   - **Completed**: Campaign has reached goal or ended successfully
   - **Cancelled**: Campaign has been cancelled and is no longer active
   - **Expected Result**: Clear understanding of each status

2. **Change status from Draft to Active**
   - Edit the campaign
   - Change **Status** dropdown to "Active"
   - Ensure all required fields are complete
   - Verify GCash configuration is set if donations are enabled
   - Save the campaign
   - **Expected Result**: Campaign becomes visible and can accept donations

3. **Pause an active campaign**
   - Edit the campaign
   - Change **Status** to "Paused"
   - Add a note in the description explaining the pause (optional)
   - Save the campaign
   - **Expected Result**: Campaign remains visible but donations are suspended

4. **Resume a paused campaign**
   - Edit the paused campaign
   - Change **Status** back to "Active"
   - Update any relevant information
   - Save the campaign
   - **Expected Result**: Campaign accepts donations again

5. **Mark campaign as completed**
   - Edit the campaign
   - Change **Status** to "Completed"
   - Verify final donation amounts are correct
   - Consider adding a thank-you message in the description
   - Save the campaign
   - **Expected Result**: Campaign is marked complete, no longer accepts donations

6. **Cancel a campaign**
   - Edit the campaign
   - Change **Status** to "Cancelled"
   - Add explanation in description if appropriate
   - Save the campaign
   - **Expected Result**: Campaign is cancelled and removed from active listings

7. **Use quick status update (if available)**
   - From the campaign list, some systems allow quick status changes
   - Click status badge or use dropdown menu
   - Confirm the status change
   - **Expected Result**: Status updates without opening full edit form

### Task 4: Monitoring Campaign Progress

This task covers tracking campaign performance and donation activity.

1. **View campaign progress from list**
   - In the campaign list, each campaign shows:
     - Current amount raised
     - Goal amount
     - Progress percentage
     - Progress bar visualization
     - Number of donors
   - **Expected Result**: Quick overview of all campaign performance

2. **Access detailed campaign view**
   - Click on a campaign name or **View** button
   - **Expected Result**: Detailed campaign page appears

3. **Review campaign statistics**
   - View detailed statistics:
     - Total amount raised
     - Number of donors
     - Progress percentage
     - Days remaining (if end date set)
     - Recent donations list
   - **Expected Result**: Comprehensive campaign performance data

4. **Check donation activity**
   - Scroll to recent donations section
   - View donor names (or "Anonymous"), amounts, and dates
   - **Expected Result**: List of recent donations for this campaign

5. **Monitor goal progress**
   - Compare current amount to goal amount
   - Check if campaign is on track to meet goal
   - Consider adjusting strategy if progress is slow
   - **Expected Result**: Understanding of campaign trajectory

6. **Review campaign timeline**
   - Check start date and end date
   - Calculate days remaining
   - Assess if timeline needs adjustment
   - **Expected Result**: Timeline awareness for planning

7. **Export campaign data (if available)**
   - Look for export or download options
   - Export donation data for reporting
   - **Expected Result**: Campaign data available for analysis

### Task 5: Deleting Campaigns

This task covers removing campaigns from the system (with important restrictions).

1. **Understand deletion restrictions**
   - Campaigns with existing donations CANNOT be deleted
   - Only campaigns with zero donations can be removed
   - This protects financial records and donor information
   - **Expected Result**: Clear understanding of deletion rules

2. **Check if campaign can be deleted**
   - View the campaign details
   - Check the number of donors/donations
   - If donations exist, campaign cannot be deleted
   - **Expected Result**: Determination of deletion eligibility

3. **Alternative to deletion for campaigns with donations**
   - Instead of deleting, change status to "Cancelled"
   - Update description to explain cancellation
   - This preserves donation records
   - **Expected Result**: Campaign is effectively removed while preserving data

4. **Delete a campaign without donations**
   - From the campaign list, click **Delete** button
   - Or from campaign detail page, click **Delete Campaign**
   - **Expected Result**: Confirmation dialog appears

5. **Confirm deletion**
   - Read the confirmation message carefully
   - Verify you're deleting the correct campaign
   - Click **Confirm** or **Yes, Delete**
   - **Expected Result**: Deletion confirmation prompt

6. **Handle deletion errors**
   - If error occurs (e.g., donations exist), read error message
   - Follow alternative approach (cancel instead of delete)
   - **Expected Result**: Error is handled appropriately

7. **Verify deletion**
   - Check that campaign no longer appears in list
   - Verify campaign is not accessible via URL
   - **Expected Result**: Campaign is completely removed

## Tips and Best Practices

- **Regular Monitoring**: Check campaign progress daily during active fundraising
- **Status Updates**: Keep campaign status current to reflect actual state
- **Goal Adjustments**: Don't change goals after donations start unless absolutely necessary
- **Transparency**: If pausing or cancelling, communicate reasons to stakeholders
- **Completion**: Mark campaigns as completed promptly when goals are reached
- **Data Preservation**: Never delete campaigns with donations; use "Cancelled" status instead
- **Featured Management**: Rotate featured campaigns to highlight different causes
- **End Dates**: Set realistic end dates and extend if needed rather than leaving open-ended
- **Progress Updates**: Post campaign updates to keep donors engaged
- **Donor Recognition**: Acknowledge donors through campaign updates

## Common Use Cases

### Use Case 1: Extending Campaign Timeline
A scholarship campaign is close to its goal but end date is approaching. Edit the campaign to extend the end date by 30 days, giving more time to reach the goal.

### Use Case 2: Pausing for Review
An infrastructure campaign needs budget review. Pause the campaign temporarily, add a note explaining the review process, and resume once approved.

### Use Case 3: Early Completion
An emergency relief campaign reaches its goal ahead of schedule. Mark as completed, add a thank-you message, and create a new campaign if additional funds are needed.

### Use Case 4: Correcting Campaign Details
After launch, you notice a typo in the campaign description. Edit the campaign, fix the error, and save without changing other settings.

## Troubleshooting

### Issue: Cannot Delete Campaign
**Symptoms**: Delete button is disabled or error message appears when attempting deletion
**Solution**: 
- Check if campaign has any donations (even pending ones)
- Campaigns with donations cannot be deleted to preserve financial records
- Use "Cancelled" status instead to remove from active listings
- Contact system administrator if you believe deletion should be allowed

### Issue: Status Change Not Reflecting
**Symptoms**: Changed campaign status but it still appears in old status
**Solution**: 
- Verify you clicked Save after changing status
- Clear browser cache and refresh the page
- Check if there are validation errors preventing save
- Verify you have permission to change campaign status

### Issue: Progress Percentage Incorrect
**Symptoms**: Progress bar or percentage doesn't match actual donations
**Solution**: 
- Progress is calculated automatically from completed donations only
- Pending donations are not included in progress
- Verify donations have been marked as "Completed" status
- Check that goal amount is correct
- System recalculates on page refresh

### Issue: Cannot Edit Campaign
**Symptoms**: Edit button is missing or edit form won't open
**Solution**: 
- Verify you have admin or staff permissions
- Check if campaign is locked by another administrator
- Try logging out and back in
- Clear browser cache
- Contact system administrator if issue persists

### Issue: GCash Configuration Cannot Be Changed
**Symptoms**: GCash dropdown is disabled or shows error
**Solution**: 
- Verify new GCash configuration is active and has QR code
- Check that "Allow Donations" is enabled
- Existing donations will keep original GCash reference
- Only future donations use new configuration
- Refresh page after updating GCash configurations
