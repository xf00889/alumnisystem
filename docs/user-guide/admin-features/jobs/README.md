# Jobs

## Overview

The Job Posting Management system allows administrators and HR staff to create, manage, and oversee job opportunities for the NORSU alumni network. This comprehensive system supports both internal NORSU positions and external organization job postings, with full application management capabilities.

## Key Features

- **Job Posting Creation**: Create detailed job postings with rich information
- **Application Management**: Review and process job applications
- **Document Requirements**: Configure required documents for internal positions
- **Status Tracking**: Monitor application progress through hiring pipeline
- **Communication Tools**: Send emails directly to applicants
- **Data Export**: Export applicant data for analysis and reporting
- **Statistics Dashboard**: Monitor job and application metrics

## Quick Start Guide

### Creating Your First Job Posting

1. Navigate to **Jobs** → **Post New Job**
2. Fill in basic information (title, company, location)
3. Add job description and requirements
4. Choose job type and source (Internal/External)
5. Configure document requirements (for internal jobs)
6. Set visibility options
7. Click **Save**

### Processing Applications

1. Go to **Jobs** → **Manage Jobs**
2. Click on the application count for a job
3. Review applications in the dashboard
4. Update statuses as you process candidates
5. Use filters to focus on specific stages
6. Send emails and add notes as needed
7. Export data for reporting

## Job Types

### Internal Jobs (NORSU Positions)
- Automatically accept applications through the system
- Can configure required documents
- Full application management features
- Ideal for NORSU faculty and staff positions

### External Jobs (Partner Organizations)
- Can link to external application portals
- Optionally accept internal applications
- Showcase opportunities from partner organizations
- Expand career options for alumni

## Application Statuses

The system tracks applications through five statuses:

1. **Pending Review**: New applications awaiting initial screening
2. **Shortlisted**: Candidates selected for further consideration
3. **Interviewed**: Candidates who have been interviewed
4. **Accepted**: Candidates offered the position
5. **Rejected**: Applications that were declined

## Access Requirements

- **User Role**: Admin or HR staff
- **Permissions**: Admin or HR group membership
- **Access Point**: Admin menu → Jobs section

## Best Practices

### Job Posting Management
- Keep job descriptions clear and concise
- Update postings regularly to keep them current
- Use featured status sparingly for high-priority positions
- Deactivate filled positions rather than deleting them
- Include salary ranges when possible

### Application Processing
- Review applications within 24-48 hours
- Use consistent evaluation criteria
- Document decisions with detailed notes
- Communicate promptly with applicants
- Maintain confidentiality of applicant information

### Communication
- Be professional and courteous in all emails
- Provide timely status updates to applicants
- Use templates for common communications
- Log all significant interactions in notes

### Data Management
- Export data regularly for backup
- Use filters to manage large application volumes
- Maintain detailed notes for compliance
- Archive completed hiring processes

## Common Workflows

### Hiring Workflow
1. Create job posting
2. Activate and optionally feature the job
3. Monitor incoming applications
4. Screen applications (Pending → Shortlisted/Rejected)
5. Schedule interviews (Shortlisted → Interviewed)
6. Make hiring decisions (Interviewed → Accepted/Rejected)
7. Communicate with all candidates
8. Deactivate job when filled

### Seasonal Hiring
1. Create job posting for seasonal position
2. Activate during hiring season
3. Process applications
4. Deactivate when season ends
5. Reactivate and update for next season

### Partner Organization Jobs
1. Create external job posting
2. Add external application link
3. Optionally enable internal applications
4. Monitor application metrics
5. Coordinate with partner organization

## Statistics and Reporting

The jobs management dashboard provides real-time statistics:

- **Active Jobs**: Currently visible job postings
- **Internal Jobs**: NORSU internal positions
- **Total Applications**: All applications across all jobs
- **Pending Applications**: Applications awaiting review

Use these metrics to:
- Monitor hiring activity
- Identify bottlenecks in the process
- Track time-to-hire
- Measure job posting effectiveness
- Report to management

## Security and Privacy

- Access restricted to admin and HR staff only
- Email communications are logged for audit purposes
- Document downloads are tracked
- Status changes are timestamped
- Exported data should be handled confidentially
