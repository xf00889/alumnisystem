# Creating Job Postings

## Overview

The job posting creation feature allows administrators and HR staff to create and publish job opportunities for the alumni network. You can create both internal NORSU positions and external job opportunities from partner organizations. Internal jobs can accept applications directly through the system, while external jobs can link to external application portals.

## Who Can Use This Feature

- **User Role**: Admin users and HR staff only
- **Permissions Required**: Admin or HR group membership
- **Prerequisites**: 
  - Active admin or HR account
  - Access to job posting management area

## How to Access

1. Log in to your admin account
2. Navigate to the **Jobs** section from the admin menu
3. Click on **Post New Job** or **Create Job Posting** button

## Key Features

- Create internal NORSU job postings
- Create external organization job postings
- Set job types (Full Time, Part Time, Contract, Remote, Internship)
- Define required skills and qualifications
- Configure required documents for internal applications
- Set salary ranges and benefits
- Mark jobs as featured
- Control job visibility (active/inactive)

## Step-by-Step Guide

### Task 1: Creating a Basic Job Posting

This task covers creating a new job posting with essential information.

1. **Access the job posting form**
   - From the Jobs management page, click **Post New Job**
   - **Expected Result**: The job posting creation form appears

2. **Enter basic job information**
   - **Job Title**: Enter the position title (e.g., "Software Engineer", "Marketing Manager")
   - **Company Name**: Enter the hiring organization name
   - **Location**: Specify the job location (city, province, or "Remote")
   - **Expected Result**: Basic information fields are populated

3. **Select job type and experience level**
   - **Job Type**: Choose from Full Time, Part Time, Contract, Remote, or Internship
   - **Experience Level**: Select Entry Level, Mid Level, Senior Level, or Executive Level
   - **Expected Result**: Job classification is set

4. **Choose source type**
   - **NORSU Internal**: For positions within NORSU (automatically accepts internal applications)
   - **External Organization**: For partner organization positions
   - **Expected Result**: Source type determines application handling

5. **Enter job description**
   - Write a comprehensive description of the position
   - Include company background and role overview
   - Use clear, professional language
   - **Expected Result**: Job description is entered in the text area

6. **Add requirements and qualifications**
   - List educational requirements
   - Specify required certifications or licenses
   - Detail minimum experience needed
   - **Expected Result**: Requirements section is completed

7. **Define responsibilities**
   - List key job duties and responsibilities
   - Describe day-to-day tasks
   - Outline reporting structure if relevant
   - **Expected Result**: Responsibilities section is filled

8. **Specify required skills**
   - Enter skills as comma-separated values
   - Include both technical and soft skills
   - Be specific (e.g., "Python, Django, REST APIs" rather than just "Programming")
   - **Expected Result**: Skills are listed for matching with alumni profiles

9. **Add benefits and salary information**
   - **Benefits**: List perks, insurance, leave policies, etc.
   - **Salary Range**: Enter compensation range (optional but recommended)
   - **Expected Result**: Compensation details are documented

10. **Set job status**
    - **Is Featured**: Check to display prominently on job board
    - **Is Active**: Check to make job visible to users
    - **Expected Result**: Job visibility settings are configured

11. **Save the job posting**
    - Click **Save** or **Create Job Posting**
    - **Expected Result**: Job is created and you're redirected to the job detail page

### Task 2: Configuring Internal Job Applications

For NORSU internal positions, you can configure the application process and required documents.

1. **Set source type to Internal**
   - Select **NORSU Internal** as the source type
   - **Expected Result**: Internal application options become available

2. **Configure required documents**
   - Scroll to the **Required Documents** section
   - Click **Add Document Requirement** to add a new document type
   - **Expected Result**: A new document requirement form appears

3. **Define document requirements**
   - **Document Name**: Enter a descriptive name (e.g., "Updated Resume")
   - **Document Type**: Select from Resume/CV, Cover Letter, Transcript, Diploma, Certification, Portfolio, Recommendation Letter, Government ID, or Other
   - **Description**: Provide specific instructions for this document
   - **Is Required**: Check if this document is mandatory
   - **File Types**: Specify allowed extensions (default: .pdf,.doc,.docx)
   - **Max File Size**: Set maximum file size in MB (default: 5MB)
   - **Expected Result**: Document requirement is configured

4. **Add multiple document requirements**
   - Repeat step 3 for each required document
   - Common combinations:
     - Resume + Cover Letter
     - Resume + Transcript + Diploma
     - Resume + Portfolio + Certifications
   - **Expected Result**: All required documents are listed

5. **Remove unwanted document requirements**
   - Check the **Delete** checkbox next to any requirement to remove
   - **Expected Result**: Document requirement is marked for deletion

6. **Save the job with document requirements**
   - Click **Save** or **Create Job Posting**
   - **Expected Result**: Job is created with configured document requirements

### Task 3: Creating External Job Postings

For external organization positions, you'll configure external application links.

1. **Set source type to External**
   - Select **External Organization** as the source type
   - **Expected Result**: External job options become available

2. **Enter application link**
   - **Application Link**: Enter the URL where applicants should apply
   - Include the full URL (e.g., https://company.com/careers/apply)
   - **Expected Result**: External application URL is saved

3. **Optional: Enable internal applications**
   - Check **Accepts Internal Applications** if you want to allow applications through the system
   - Leave unchecked if applicants should only use the external link
   - **Expected Result**: Application method is configured

4. **Complete remaining job details**
   - Fill in all other job information as described in Task 1
   - **Expected Result**: External job posting is complete

5. **Save the external job posting**
   - Click **Save** or **Create Job Posting**
   - **Expected Result**: External job is created and visible on the job board

### Task 4: Setting Job Categories and Advanced Options

Configure additional job settings for better organization and discoverability.

1. **Select job category**
   - Choose the most appropriate category:
     - Technology
     - Finance & Accounting
     - Healthcare & Medical
     - Education & Training
     - Sales & Marketing
     - Hospitality & Tourism
     - Manufacturing & Production
     - Administrative & Clerical
     - Construction & Engineering
     - Creative & Design
     - Other
   - **Expected Result**: Job is categorized for filtering

2. **Configure featured status**
   - Check **Is Featured** to display the job prominently
   - Featured jobs appear at the top of the job board
   - Limit featured jobs to the most important positions
   - **Expected Result**: Job is marked as featured

3. **Set initial visibility**
   - **Is Active**: Check to make job immediately visible
   - Uncheck to save as draft for later publication
   - **Expected Result**: Job visibility is controlled

4. **Review and save**
   - Review all entered information
   - Click **Save** or **Create Job Posting**
   - **Expected Result**: Job is created with all advanced settings

## Tips and Best Practices

- **Be Specific**: Use clear, specific job titles that accurately reflect the position
- **Complete Information**: Fill in all relevant fields to provide comprehensive job details
- **Skills Matching**: Enter detailed skills to enable better matching with alumni profiles
- **Regular Updates**: Keep job postings current and remove or deactivate filled positions
- **Featured Jobs**: Use featured status sparingly for high-priority positions
- **Document Requirements**: For internal jobs, only require essential documents to avoid deterring applicants
- **External Links**: Always test external application links before publishing
- **Salary Transparency**: Including salary ranges increases application rates
- **Mobile-Friendly**: Write concise descriptions that are easy to read on mobile devices

## Common Use Cases

### Use Case 1: NORSU Faculty Position
Create an internal job posting for a faculty position requiring transcript, diploma, and teaching certifications. Set as featured to attract qualified alumni.

### Use Case 2: Partner Company Opportunity
Post an external job from a partner organization with a link to their application portal. Include detailed requirements to help alumni assess fit before applying externally.

### Use Case 3: Remote Internship
Create a remote internship posting with flexible requirements to provide opportunities for recent graduates. Set lower experience requirements and emphasize learning opportunities.

### Use Case 4: Draft Job Posting
Create a job posting with "Is Active" unchecked to save as a draft. Review with hiring managers before activating and making visible to alumni.

## Troubleshooting

### Issue: Cannot Save Job Posting
**Symptoms**: Save button doesn't work or error message appears
**Solution**: 
- Ensure all required fields are filled (Job Title, Company Name, Location, Job Description)
- For external jobs, provide either an Application Link or check "Accepts Internal Applications"
- Check that file size limits for document requirements are reasonable (under 10MB)

### Issue: Document Requirements Not Showing
**Symptoms**: Required documents section is not visible
**Solution**: 
- Ensure Source Type is set to "NORSU Internal"
- Document requirements only apply to internal positions
- For external jobs accepting internal applications, add document requirements after setting the checkbox

### Issue: Job Not Appearing on Job Board
**Symptoms**: Created job doesn't show up in job listings
**Solution**: 
- Verify "Is Active" checkbox is checked
- Check that the job was saved successfully
- Clear browser cache and refresh the job board
- Verify you're viewing the correct job type filter

### Issue: Application Link Not Working
**Symptoms**: External application link returns an error
**Solution**: 
- Ensure the URL includes http:// or https://
- Test the link in a new browser tab before saving
- Verify the external company's application portal is active
- Contact the external organization if their link is broken

## Additional Notes

- Job postings are automatically assigned a unique slug based on the job title and company name
- The system prevents duplicate slugs by adding numeric suffixes if needed
- Internal jobs automatically set "Accepts Internal Applications" to true
- Skills entered in job postings are used for skill-based job matching with alumni profiles
- Job posting creation is logged for audit purposes
- You can save jobs as inactive drafts and activate them later
