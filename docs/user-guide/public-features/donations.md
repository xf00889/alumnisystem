# Donations

## Overview

The Public Donations feature allows anyone to support NORSU alumni initiatives and campaigns without requiring registration. Visitors can browse active donation campaigns, make contributions, and upload payment proof to complete their donations.

## Who Can Use This Feature

- **All visitors** (no authentication required)
- Anyone who wants to support NORSU alumni initiatives
- Both registered and non-registered users

## Prerequisites

- None - this feature is publicly accessible
- Internet connection and web browser
- Payment method (for making donations)
- Payment proof (receipt/screenshot) for verification

## How to Access

1. Navigate to the NORSU Alumni System website
2. Click on "Donations" or "Campaigns" in the main navigation menu
3. Or visit: `/donations/campaigns/`

## Key Features

- Browse all active public donation campaigns
- View campaign details and progress
- Make donations without registration
- Upload payment proof
- Track donation progress
- Filter and search campaigns
- View recent donors (non-anonymous)
- See campaign updates

## Step-by-Step Guide

### Task 1: Browse Donation Campaigns

1. Access the campaigns page from the main navigation
   - **Expected Result**: You'll see a list of active public campaigns

2. Review the campaign cards
   - Each campaign card displays:
     - Campaign name
     - Campaign image
     - Short description
     - Goal amount
     - Current amount raised
     - Progress bar
     - Number of donors
     - Campaign type
     - End date
   - **Expected Result**: Campaigns are displayed in an organized grid layout

3. Use filters to find specific campaigns
   - Filter by campaign type
   - Filter by status (active, ending soon)
     - Sort by: Recent, Ending Soon, Progress, Goal Amount
   - **Expected Result**: Campaign list updates based on selected filters

4. Search for campaigns
   - Enter keywords in the search box
   - Search looks through campaign names and descriptions
   - **Expected Result**: Only matching campaigns are displayed

### Task 2: View Campaign Details

1. Click on a campaign card to view full details
   - **Expected Result**: Campaign detail page opens

2. Review complete campaign information
   - The detail page shows:
     - Full campaign name
     - Complete description
     - Campaign type
     - Goal amount
     - Current amount raised
     - Progress percentage
     - Start and end dates
     - Beneficiaries
     - Campaign status
     - Recent donations list
     - Campaign updates
     - Donation statistics
   - **Expected Result**: All campaign information is clearly displayed

3. View recent donors
   - See list of recent non-anonymous donations
   - Anonymous donations show as "Anonymous Donor"
   - **Expected Result**: Recent donor list is visible with donation amounts

4. Read campaign updates
   - Campaign organizers may post updates
   - Updates show creation date and content
   - **Expected Result**: Updates are displayed chronologically

### Task 3: Make a Donation (Non-Registered Users)

1. On the campaign detail page, locate the donation form
   - **Expected Result**: Donation form is visible on the page

2. Fill in your donation information
   - **Required fields**:
     - Your full name
     - Your email address
     - Donation amount
   - **Optional fields**:
     - Personal message
     - Make donation anonymous (checkbox)
   - **Expected Result**: Form fields are ready for input

3. Enter the donation amount
   - Minimum amount may be specified
   - Enter amount in the local currency
   - **Expected Result**: Amount is validated and accepted

4. Choose anonymity option
   - Check "Make this donation anonymous" if you want to remain anonymous
   - Anonymous donations won't show your name publicly
   - **Expected Result**: Checkbox is selected/unselected as desired

5. Add a personal message (optional)
   - Write a message of support or encouragement
   - **Expected Result**: Message is entered in the text area

6. Review your donation details
   - Double-check all information
   - **Expected Result**: All fields are correctly filled

7. Click "Proceed to Payment Instructions"
   - **Expected Result**: You're redirected to the payment instructions page

### Task 4: Complete Payment and Upload Proof

1. Review the payment instructions page
   - **Information displayed**:
     - GCash QR code for payment
     - GCash account details
     - Donation amount
     - Reference number
     - Payment instructions
   - **Expected Result**: Payment details are clearly shown

2. Make the payment using GCash
   - Open your GCash app
   - Scan the QR code OR enter the account details manually
   - Enter the exact donation amount
   - Complete the payment
   - **Expected Result**: Payment is processed through GCash

3. Take a screenshot or save your payment receipt
   - Capture the GCash payment confirmation
   - Ensure the reference number is visible
   - **Expected Result**: You have proof of payment ready

4. Upload your payment proof
   - Click "Choose File" or "Upload Payment Proof"
   - Select your screenshot/receipt image
   - Supported formats: JPG, PNG, PDF
   - Maximum file size: Usually 5MB
   - **Expected Result**: File is selected and ready to upload

5. Submit the payment proof
   - Click "Submit Payment Proof" button
   - **Expected Result**: Upload is processed

6. Receive confirmation
   - You'll see a success message
   - An email confirmation will be sent to your provided email
   - **Expected Result**: Donation is recorded as "Pending Verification"

### Task 5: Track Your Donation

1. Save your donation confirmation page
   - Bookmark the URL or save the reference number
   - **Expected Result**: You can return to check donation status

2. Check your email for confirmation
   - Email includes:
     - Donation details
     - Reference number
     - Campaign information
     - Next steps
   - **Expected Result**: Confirmation email is received

3. Wait for verification
   - Admin will verify your payment proof
   - This usually takes 1-3 business days
   - **Expected Result**: Donation status changes to "Completed" after verification

4. Receive receipt (after verification)
   - Once verified, you'll receive an official receipt via email
   - **Expected Result**: Receipt email is sent to your address

## Tips and Best Practices

- **Use Correct Amount**: Enter the exact amount you plan to donate
- **Keep Payment Proof**: Save your GCash receipt until donation is verified
- **Provide Valid Email**: Use an email you check regularly for updates
- **Check Spam Folder**: Confirmation emails might go to spam
- **Note Reference Number**: Save your donation reference number for tracking
- **Upload Clear Images**: Ensure payment proof is clear and readable
- **Be Patient**: Verification may take 1-3 business days
- **Contact Support**: If verification takes longer, contact the admin

## Understanding Donation Status

- **Pending**: Donation created, awaiting payment proof
- **Pending Verification**: Payment proof uploaded, awaiting admin verification
- **Completed**: Donation verified and counted toward campaign goal
- **Failed**: Payment could not be verified
- **Refunded**: Donation was refunded

## Troubleshooting

### Issue: Cannot see donation form
**Solution**:
- Ensure GCash payment is configured by admin
- Refresh the page
- Try a different browser
- Check if campaign is still active

### Issue: Payment proof upload fails
**Solution**:
- Check file size (must be under 5MB)
- Verify file format (JPG, PNG, or PDF)
- Ensure stable internet connection
- Try compressing the image
- Try a different browser

### Issue: Did not receive confirmation email
**Solution**:
- Check your spam/junk folder
- Verify you entered the correct email address
- Wait a few minutes and check again
- Contact support with your reference number

### Issue: Donation not verified after several days
**Solution**:
- Check your email for any requests for additional information
- Ensure your payment proof was clear and complete
- Contact support with your reference number
- Provide additional proof if requested

### Issue: Made wrong donation amount
**Solution**:
- Contact support immediately with your reference number
- Provide details of the error
- Admin can help resolve the issue

### Issue: Cannot access payment instructions page
**Solution**:
- Check your internet connection
- Clear browser cache
- Try a different browser
- Ensure you completed the donation form correctly

## Privacy and Security

- **Email Privacy**: Your email is only used for donation confirmation and receipts
- **Anonymous Donations**: Your name won't appear publicly if you choose anonymous
- **Payment Security**: Payments are processed through GCash's secure platform
- **Data Protection**: Your information is stored securely and not shared with third parties
