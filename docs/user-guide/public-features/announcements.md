# Announcements

## Overview

The Public Announcements feature allows visitors to view important announcements and updates from the NORSU Alumni System without requiring registration or login. This feature provides easy access to system-wide news, events, and important information for the entire alumni community.

## Who Can Use This Feature

- **All visitors** (no authentication required)
- Anyone with internet access can view public announcements

## Prerequisites

- None - this feature is publicly accessible
- Internet connection and web browser

## How to Access

1. Navigate to the NORSU Alumni System website
2. Click on "Announcements" in the main navigation menu
3. Or visit: `/announcements/public/`

## Key Features

- Browse all active public announcements
- Search announcements by keywords
- Filter announcements by category
- View detailed announcement information
- Paginated list for easy navigation
- View count tracking

## Step-by-Step Guide

### Task 1: Browse Public Announcements

1. Access the public announcements page from the main navigation
   - **Expected Result**: You'll see a list of all active public announcements ordered by date posted (newest first)

2. Scroll through the announcement list
   - Each announcement card displays:
     - Title
     - Category badge
     - Priority level indicator
     - Date posted
     - Brief content preview
     - View count
   - **Expected Result**: Announcements are displayed in a clean, card-based layout

3. Use pagination controls at the bottom of the page
   - Click page numbers to navigate through multiple pages
   - 10 announcements are displayed per page
   - **Expected Result**: Page loads with the selected page of announcements

### Task 2: Search for Announcements

1. Locate the search box at the top of the announcements page
   - **Expected Result**: Search input field is visible and ready for input

2. Enter keywords related to the announcement you're looking for
   - Search looks through:
     - Announcement titles
     - Announcement content
   - **Expected Result**: Search results update dynamically as you type

3. Press Enter or click the search button
   - **Expected Result**: Page displays only announcements matching your search query

4. Clear the search to return to all announcements
   - Click the "Clear" button or remove text from search box
   - **Expected Result**: Full announcement list is restored

### Task 3: Filter Announcements by Category

1. Look for the category filter section (usually in a sidebar or filter panel)
   - **Expected Result**: List of available categories is displayed

2. Click on a category to filter announcements
   - Available categories may include:
     - General
     - Events
     - Academic
     - Career
     - Alumni News
   - **Expected Result**: Only announcements in the selected category are displayed

3. Select multiple categories if supported
   - Check multiple category checkboxes
   - **Expected Result**: Announcements from any of the selected categories are shown

4. Clear category filters
   - Uncheck all categories or click "Clear Filters"
   - **Expected Result**: All announcements are displayed again

### Task 4: View Announcement Details

1. Click on any announcement card or title
   - **Expected Result**: You're taken to the announcement detail page

2. Read the full announcement content
   - The detail page shows:
     - Full title
     - Complete content
     - Category
     - Priority level
     - Date posted
     - Last modified date
     - View count
   - **Expected Result**: All announcement information is clearly displayed

3. Return to the announcement list
   - Click the "Back to Announcements" button or browser back button
   - **Expected Result**: You return to the announcements list page

## Tips and Best Practices

- **Use Search Effectively**: Enter specific keywords to quickly find relevant announcements
- **Check Regularly**: Visit the announcements page frequently to stay updated on important news
- **Note Priority Levels**: Pay attention to "URGENT" or "HIGH" priority announcements
- **Bookmark Important Announcements**: Save the URL of important announcements for future reference
- **Register for More**: Consider registering for an account to access additional features and personalized announcements

## Understanding Priority Levels

Announcements are marked with different priority levels:

- **URGENT** (Red badge): Requires immediate attention
- **HIGH** (Orange badge): Important information
- **MEDIUM** (Blue badge): Standard announcements
- **LOW** (Gray badge): General information

## Troubleshooting

### Issue: No announcements are displayed
**Solution**: 
- Check your internet connection
- Refresh the page
- Clear your browser cache
- Verify you're on the correct URL

### Issue: Search returns no results
**Solution**:
- Check your spelling
- Try different keywords
- Use broader search terms
- Clear filters and try again

### Issue: Page loads slowly
**Solution**:
- Check your internet connection speed
- Try refreshing the page
- Clear browser cache and cookies
- Try a different browser

### Issue: Cannot view announcement details
**Solution**:
- Ensure the announcement is still active
- Try clicking the announcement title instead of the card
- Refresh the page and try again
- Check if the announcement has been removed
