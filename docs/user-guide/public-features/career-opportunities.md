# Career Opportunities

## Overview

The Career Opportunities feature provides public access to job postings and career opportunities shared through the NORSU Alumni System. Visitors can browse job listings, search for positions, and view detailed job information without requiring registration.

## Who Can Use This Feature

- **All visitors** (no authentication required)
- Job seekers from the NORSU alumni community
- Anyone interested in career opportunities

## Prerequisites

- None - this feature is publicly accessible
- Internet connection and web browser

## How to Access

1. Navigate to the NORSU Alumni System website
2. Click on "Careers" or "Jobs" in the main navigation menu
3. Or visit: `/jobs/careers/`

## Key Features

- Browse all active job postings
- View featured job opportunities
- Search jobs by keywords
- Filter by job type, location, and category
- View detailed job descriptions
- See job requirements and qualifications
- Access application links for external jobs
- View job posting dates

## Step-by-Step Guide

### Task 1: Browse Job Listings

1. Access the careers page from the main navigation
   - **Expected Result**: You'll see the careers landing page with featured and recent jobs

2. Review featured jobs section
   - Up to 3 featured jobs are highlighted at the top
   - Featured jobs are marked with a special badge
   - **Expected Result**: Featured jobs are prominently displayed

3. Browse recent job postings
   - Recent jobs are listed below featured jobs
   - Up to 10 recent jobs are shown initially
   - **Expected Result**: Job listings are displayed in a clean, organized format

4. View job statistics
   - Total number of active jobs
   - Number of featured jobs
   - Number of recent postings
   - **Expected Result**: Statistics are displayed on the page

### Task 2: Search for Jobs

1. Locate the search box on the careers page
   - **Expected Result**: Search input field is visible

2. Enter keywords to search for jobs
   - Search looks through:
     - Job titles
     - Company names
     - Job descriptions
   - **Expected Result**: Search results update based on your query

3. Press Enter or click the search button
   - **Expected Result**: Only jobs matching your search criteria are displayed

4. Clear the search
   - Remove text from search box or click "Clear"
   - **Expected Result**: Full job list is restored

### Task 3: Filter Job Listings

1. Use the job type filter
   - Available options:
     - Full-time
     - Part-time
     - Contract
     - Internship
     - Freelance
   - **Expected Result**: Jobs are filtered by selected type

2. Filter by location
   - Enter or select a location
   - Jobs are filtered by location field
   - **Expected Result**: Only jobs in the specified location are shown

3. Filter by category
   - Available categories may include:
     - Technology
     - Education
     - Healthcare
     - Business
     - Engineering
     - And more
   - **Expected Result**: Jobs in selected category are displayed

4. Combine multiple filters
   - Use job type + location + category together
   - **Expected Result**: Jobs matching all selected criteria are shown

5. Clear all filters
   - Click "Clear Filters" or remove individual filters
   - **Expected Result**: Full job list is restored

### Task 4: View Job Details

1. Click on any job card or job title
   - **Expected Result**: Job details modal or page opens

2. Review complete job information
   - **Job Details Include**:
     - Job title
     - Company name
     - Location
     - Job type (Full-time, Part-time, etc.)
     - Category
     - Posted date
     - Job description
     - Key responsibilities
     - Requirements and qualifications
     - Experience level required
     - Skills required
     - Education requirements
     - Benefits (if provided)
     - Salary range (if provided)
   - **Expected Result**: All job information is clearly displayed

3. Check the source type
   - **Internal Jobs**: Posted by NORSU alumni system (requires registration to apply)
   - **External Jobs**: Posted from external sources (application link provided)
   - **Expected Result**: Source type is clearly indicated

4. Note if the job is featured
   - Featured jobs have a special badge or indicator
   - **Expected Result**: Featured status is visible

### Task 5: Apply for Jobs

#### For External Jobs:

1. Locate the "Apply Now" or "Application Link" button
   - **Expected Result**: Button is visible on the job details page

2. Click the application link
   - **Expected Result**: You're redirected to the external application website

3. Complete the application on the external site
   - Follow the employer's application process
   - **Expected Result**: You can apply directly on the employer's website

#### For Internal Jobs (Registration Required):

1. Note that internal jobs require registration
   - A message indicates "Please log in or register to apply"
   - **Expected Result**: Clear message about registration requirement

2. Click "Register" or "Login" button
   - **Expected Result**: You're directed to the registration/login page

3. Complete registration (see [Registration Process](registration.md))
   - **Expected Result**: After registration, you can apply for internal jobs

## Tips and Best Practices

- **Check Regularly**: New jobs are posted frequently - visit often
- **Use Specific Keywords**: Be specific in your search to find relevant positions
- **Read Full Descriptions**: Review all job details before applying
- **Note Application Deadlines**: Some jobs may have closing dates
- **Bookmark Interesting Jobs**: Save job URLs to review later
- **Register for More Features**: Create an account to:
  - Apply for internal jobs
  - Track your applications
  - Get skill-based job recommendations
  - Receive job alerts

## Understanding Job Information

### Job Types
- **Full-time**: Regular full-time employment
- **Part-time**: Part-time employment
- **Contract**: Fixed-term contract position
- **Internship**: Internship or training position
- **Freelance**: Project-based or freelance work

### Experience Levels
- **Entry Level**: 0-2 years of experience
- **Mid Level**: 3-5 years of experience
- **Senior Level**: 5+ years of experience
- **Executive**: Leadership/management positions

### Job Categories
Jobs are organized into categories such as:
- Technology/IT
- Education
- Healthcare
- Business/Management
- Engineering
- Marketing
- Finance
- And more

### Source Types
- **Internal**: Posted by NORSU alumni system (requires registration to apply)
- **External**: Posted from external sources (apply via external link)
- **Scraped**: Automatically collected from job boards

## Troubleshooting

### Issue: No jobs are displayed
**Solution**:
- There may be no active job postings at this time
- Check back later for new postings
- Clear any active filters
- Refresh the page
- Verify your internet connection

### Issue: Search returns no results
**Solution**:
- Check your spelling
- Try different or broader keywords
- Clear filters and search again
- Use more general terms

### Issue: Cannot view job details
**Solution**:
- Ensure the job is still active
- Refresh the page
- Try clicking the job title directly
- Clear browser cache
- Try a different browser

### Issue: Application link not working
**Solution**:
- Verify the link is correct
- Try copying and pasting the URL
- Check if the external site is accessible
- Contact the employer directly if link is broken
- Report broken links to system administrators

### Issue: Cannot apply for internal jobs
**Solution**:
- Internal jobs require registration
- Click "Register" to create an account
- Complete the registration process
- Return to the job and apply after logging in

### Issue: Filters not working
**Solution**:
- Refresh the page
- Clear browser cache
- Try applying filters one at a time
- Use a different browser
