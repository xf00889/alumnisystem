# Events

## Overview

The Public Events feature allows visitors to browse and view upcoming events organized by the NORSU Alumni System without requiring registration or login. This feature helps the community stay informed about alumni gatherings, reunions, networking events, and other activities.

## Who Can Use This Feature

- **All visitors** (no authentication required)
- Anyone interested in NORSU alumni events

## Prerequisites

- None - this feature is publicly accessible
- Internet connection and web browser

## How to Access

1. Navigate to the NORSU Alumni System website
2. Click on "Events" in the main navigation menu
3. Or visit: `/events/public/`

## Key Features

- Browse all published upcoming events
- Search events by keywords
- View detailed event information
- See event dates, times, and locations
- View RSVP counts (attending, maybe, not attending)
- Filter events by various criteria
- Paginated event listings

## Step-by-Step Guide

### Task 1: Browse Public Events

1. Access the public events page from the main navigation
   - **Expected Result**: You'll see a list of all published upcoming events ordered by start date

2. Review the event cards displayed
   - Each event card shows:
     - Event title
     - Event image (if available)
     - Start date and time
     - End date and time
     - Location
     - Event status (Published)
     - Brief description
     - RSVP counts
   - **Expected Result**: Events are displayed in an organized, card-based layout

3. Navigate through multiple pages using pagination
   - 12 events are displayed per page
   - Click page numbers at the bottom to view more events
   - **Expected Result**: Additional events load on each page

### Task 2: Search for Events

1. Locate the search box on the events page
   - **Expected Result**: Search input field is visible at the top of the page

2. Enter keywords to search for specific events
   - Search looks through:
     - Event titles
     - Event descriptions
     - Event locations
   - **Expected Result**: Search results filter the event list in real-time

3. Press Enter or click the search button
   - **Expected Result**: Only events matching your search criteria are displayed

4. Clear the search
   - Remove text from the search box or click "Clear"
   - **Expected Result**: Full event list is restored

### Task 3: View Event Details

1. Click on any event card or event title
   - **Expected Result**: You're taken to the event detail page

2. Review the complete event information
   - The detail page displays:
     - Full event title
     - Complete description
     - Start date and time
     - End date and time
     - Location address
     - Virtual event link (if applicable)
     - Event status
     - Maximum participants (if set)
     - Event creator information
     - RSVP statistics:
       - Number attending
       - Number not attending
       - Number of maybes
   - **Expected Result**: All event details are clearly presented

3. Note virtual event information
   - If the event is virtual, a virtual meeting link will be displayed
   - **Expected Result**: Virtual link is visible for online events

4. Return to the events list
   - Click "Back to Events" or use the browser back button
   - **Expected Result**: You return to the events listing page

### Task 4: Understand Event Status

1. Check the event status indicator
   - **Published**: Event is active and visible to the public
   - **Expected Result**: Only published events appear in the public view

2. Note the event timing
   - Events shown are upcoming (start date is in the future)
   - Past events are not displayed in the public view
   - **Expected Result**: Only future events are listed

## Tips and Best Practices

- **Check Event Dates**: Pay attention to start and end dates to plan your attendance
- **Note Virtual vs. In-Person**: Look for the virtual event indicator to know if you can attend remotely
- **Register for RSVP**: Create an account to RSVP to events and receive updates
- **Bookmark Events**: Save event URLs to easily return to event details
- **Check Regularly**: Visit the events page frequently as new events are added regularly
- **Note Maximum Participants**: Some events have limited capacity - register early if interested

## Understanding Event Information

### Event Types
- **In-Person Events**: Physical location provided
- **Virtual Events**: Online meeting link provided
- **Hybrid Events**: Both physical location and virtual link available

### RSVP Counts
- **Attending**: Number of registered users who confirmed attendance
- **Maybe**: Number of users who might attend
- **Not Attending**: Number of users who declined

### Event Status
- **Published**: Event is active and visible
- **Draft**: Not visible to public (admin only)
- **Cancelled**: Event has been cancelled
- **Completed**: Event has ended

## Troubleshooting

### Issue: No events are displayed
**Solution**:
- There may be no upcoming published events at this time
- Check back later for new events
- Verify your internet connection
- Refresh the page

### Issue: Search returns no results
**Solution**:
- Check your spelling
- Try different or broader keywords
- Clear the search and browse all events
- Verify events exist for your search criteria

### Issue: Cannot view event details
**Solution**:
- Ensure the event is still published
- Refresh the page
- Try clicking the event title directly
- Check if the event has been removed or cancelled

### Issue: Virtual event link not working
**Solution**:
- Note: Virtual event links are only accessible to registered users who have RSVP'd
- Register for an account to access virtual event links
- Contact the event organizer if you're registered but cannot access the link

### Issue: Page loads slowly
**Solution**:
- Check your internet connection
- Clear browser cache
- Try a different browser
- Refresh the page

## Limitations for Public View

As a non-registered visitor, you cannot:
- RSVP to events
- Access virtual event meeting links
- Receive event notifications
- View your event history
- Contact event organizers directly through the system

**To access these features**: [Register for an account](registration.md)
