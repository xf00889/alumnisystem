# Public Features

## Overview

This section covers all features available to visitors without requiring registration or login. These features allow anyone to explore the NORSU Alumni System and access important information about the alumni community.

## Available Public Features

### 1. [Public Announcements](announcements.md)
View system-wide announcements and important updates without registration.

**Key Capabilities:**
- Browse all active public announcements
- Search announcements by keywords
- Filter by category
- View announcement details
- Track announcement views

**Access:** `/announcements/public/`

---

### 2. [Public Events](events.md)
Discover upcoming alumni events and gatherings open to the public.

**Key Capabilities:**
- Browse published upcoming events
- Search events by keywords
- View event details and schedules
- See RSVP counts
- Access virtual event information

**Access:** `/events/public/`

---

### 3. [Public Donations](donations.md)
Support NORSU alumni initiatives through donations without requiring an account.

**Key Capabilities:**
- Browse active donation campaigns
- View campaign progress and goals
- Make donations as a guest
- Upload payment proof
- Track donation status

**Access:** `/donations/campaigns/`

---

### 4. [Career Opportunities](career-opportunities.md)
Explore job postings and career opportunities from the alumni network.

**Key Capabilities:**
- Browse all active job listings
- Search and filter jobs
- View detailed job descriptions
- Access application links
- See featured opportunities

**Access:** `/jobs/careers/`

---

### 5. [Registration Process](registration.md)
Create an account to access the full range of alumni system features.

**Key Capabilities:**
- Email-based registration
- Email verification
- Profile completion
- Account activation
- Secure password setup

**Access:** `/accounts/signup/`

---

## Common Questions

### Do I need to register to view content?
No, you can view announcements, events, job listings, and donation campaigns without registration. However, registration is required to interact with these features (RSVP, apply, donate with tracking, etc.).

### Is registration free?
Yes, registration is completely free for all NORSU alumni and community members.

### What information do I need to register?
You'll need a valid email address and basic personal information. See the [Registration Process](registration.md) for details.

### Can I donate without registering?
Yes, you can make donations as a guest. However, registering allows you to track your donation history and receive official receipts.

### Can I apply for jobs without an account?
External jobs with application links can be accessed without registration. Internal jobs require registration to apply.

### How do I get help?
- Check the troubleshooting sections in each feature guide
- Contact support through the website
- Email the alumni office

## Privacy & Security

All public features are designed with privacy and security in mind:

- **No Personal Data Required**: Browse without providing personal information
- **Secure Connections**: All pages use HTTPS encryption
- **Optional Registration**: Choose when you're ready to create an account
- **Data Protection**: Your information is protected according to privacy policies

## Technical Requirements

To use public features, you need:

- **Web Browser**: Modern browser (Chrome, Firefox, Safari, Edge)
- **Internet Connection**: Stable internet connection
- **JavaScript Enabled**: For interactive features
- **Cookies Enabled**: For session management

## Accessibility

The NORSU Alumni System is designed to be accessible to all users:

- Screen reader compatible
- Keyboard navigation support
- High contrast mode available
- Responsive design for mobile devices
- Clear and readable fonts
