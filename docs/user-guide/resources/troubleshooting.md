# Troubleshooting

This guide helps you resolve common issues with the NORSU Alumni System.

## Login & Authentication Issues

### Issue: Cannot log in with correct credentials
**Symptoms**: Error message when entering username and password
**Solutions**:
1. Verify your email address is correct
2. Check if Caps Lock is on
3. Try resetting your password
4. Clear your browser cache and cookies
5. Ensure your account has been verified

### Issue: Verification email not received
**Symptoms**: No verification email in inbox
**Solutions**:
1. Check your spam/junk folder
2. Wait a few minutes for email delivery
3. Request a new verification email
4. Verify the email address you provided is correct
5. Check if your email provider is blocking emails

### Issue: Password reset link expired
**Symptoms**: Error when clicking password reset link
**Solutions**:
1. Request a new password reset link
2. Use the link within the specified time frame (usually 24 hours)
3. Check that you're using the most recent reset email

## Profile & Account Issues

### Issue: Cannot upload profile photo
**Symptoms**: Error message when uploading image
**Solutions**:
1. Check file size (must be under the maximum limit)
2. Verify file format (JPG, PNG, GIF supported)
3. Try a different image
4. Ensure stable internet connection
5. Try using a different browser

### Issue: Changes not saving
**Symptoms**: Profile updates don't persist
**Solutions**:
1. Ensure all required fields are filled
2. Check for error messages on the form
3. Verify internet connection is stable
4. Try refreshing the page and re-entering information
5. Clear browser cache

### Issue: Cannot delete education/experience entry
**Symptoms**: Delete button not working
**Solutions**:
1. Refresh the page and try again
2. Ensure you have permission to delete
3. Check if the entry is marked as primary (may need to set another as primary first)

## Connection & Networking Issues

### Issue: Connection request not sending
**Symptoms**: No confirmation after sending request
**Solutions**:
1. Verify you're not already connected
2. Check if you have a pending request
3. Ensure the user hasn't blocked connection requests
4. Refresh the page and try again

### Issue: Cannot see connection's profile
**Symptoms**: Profile page shows limited information
**Solutions**:
1. Verify you're connected with the user
2. Check the user's privacy settings
3. Ensure you're logged in

## Messaging Issues

### Issue: Messages not sending
**Symptoms**: Message appears stuck or shows error
**Solutions**:
1. Check internet connection
2. Verify you're connected with the recipient
3. Check message length (may have character limit)
4. Try refreshing the page
5. Ensure recipient hasn't blocked you

### Issue: Cannot upload file attachment
**Symptoms**: File upload fails
**Solutions**:
1. Check file size limit
2. Verify file type is allowed
3. Ensure stable internet connection
4. Try a different file

### Issue: Not receiving message notifications
**Symptoms**: No alerts for new messages
**Solutions**:
1. Check notification settings in your account
2. Verify browser notification permissions
3. Check if notifications are enabled for the site
4. Try logging out and back in

## Event Issues

### Issue: Cannot RSVP to event
**Symptoms**: RSVP button not working
**Solutions**:
1. Verify event is still accepting RSVPs
2. Check if event has reached capacity
3. Ensure you meet event requirements
4. Refresh the page and try again

### Issue: Event not showing in "My Events"
**Symptoms**: RSVP'd event not appearing
**Solutions**:
1. Verify your RSVP was confirmed
2. Check the event date (past events may be in different section)
3. Refresh the page
4. Check filter settings

## Job Board Issues

### Issue: Cannot submit job application
**Symptoms**: Application form shows error
**Solutions**:
1. Ensure all required fields are completed
2. Verify uploaded documents meet requirements
3. Check file sizes and formats
4. Ensure stable internet connection
5. Try submitting again after refreshing

### Issue: Application status not updating
**Symptoms**: Status shows as pending for long time
**Solutions**:
1. This is normal - wait for admin review
2. Check back periodically for updates
3. Contact the job poster if urgent

## Donation Issues

### Issue: Cannot upload payment proof
**Symptoms**: Upload fails or shows error
**Solutions**:
1. Check file size and format
2. Ensure image is clear and readable
3. Verify internet connection
4. Try a different file format
5. Contact admin if problem persists

### Issue: Donation not showing in history
**Symptoms**: Recent donation not appearing
**Solutions**:
1. Wait for admin verification
2. Check if payment proof was uploaded successfully
3. Verify donation was submitted (check confirmation)
4. Contact admin if missing after verification

## Group Issues

### Issue: Cannot join group
**Symptoms**: Join button not working or shows error
**Solutions**:
1. Check if group requires approval
2. Verify you answered security questions correctly
3. Ensure you're not already a member
4. Check if group is at capacity

### Issue: Cannot post in group
**Symptoms**: Post button disabled or shows error
**Solutions**:
1. Verify you're a group member
2. Check group posting permissions
3. Ensure post meets group guidelines
4. Try refreshing the page

## Search & Filter Issues

### Issue: Search returns no results
**Symptoms**: No results when searching
**Solutions**:
1. Check spelling and try different keywords
2. Remove some filters to broaden search
3. Try searching with partial terms
4. Verify data exists for your search criteria

### Issue: Filters not working
**Symptoms**: Results don't match selected filters
**Solutions**:
1. Clear all filters and reapply
2. Refresh the page
3. Try different filter combinations
4. Clear browser cache

## Performance Issues

### Issue: Page loading slowly
**Symptoms**: Long wait times for pages to load
**Solutions**:
1. Check your internet connection speed
2. Clear browser cache and cookies
3. Try a different browser
4. Disable browser extensions temporarily
5. Check if issue occurs on different device

### Issue: Images not loading
**Symptoms**: Broken image icons or blank spaces
**Solutions**:
1. Refresh the page
2. Check internet connection
3. Clear browser cache
4. Try a different browser
5. Verify images exist (may have been removed)

## Browser Compatibility Issues

### Issue: Features not working in browser
**Symptoms**: Buttons, forms, or features don't work
**Solutions**:
1. Update your browser to the latest version
2. Try a different browser (Chrome, Firefox, Safari, Edge)
3. Enable JavaScript in browser settings
4. Disable browser extensions that might interfere
5. Clear browser cache and cookies

### Recommended Browsers
- Google Chrome (latest version)
- Mozilla Firefox (latest version)
- Safari (latest version)
- Microsoft Edge (latest version)

## Mobile Issues

### Issue: Site not displaying correctly on mobile
**Symptoms**: Layout issues or features not accessible
**Solutions**:
1. Rotate device to landscape mode
2. Zoom in/out to adjust view
3. Try desktop mode in browser settings
4. Update mobile browser
5. Use desktop version if mobile issues persist

## Admin-Specific Issues

### Issue: Cannot access admin features
**Symptoms**: Admin menu not visible
**Solutions**:
1. Verify you have admin permissions
2. Log out and log back in
3. Contact system administrator to verify role
4. Clear browser cache

### Issue: Cannot approve/reject applications
**Symptoms**: Action buttons not working
**Solutions**:
1. Verify you have appropriate permissions
2. Check if application is already processed
3. Refresh the page
4. Try a different browser

## Still Having Issues?

If you've tried the solutions above and still experience problems:

1. **Document the Issue**: Note what you were doing when the problem occurred
2. **Take Screenshots**: Capture any error messages
3. **Contact Support**: Use the feedback form or contact system administrators
4. **Provide Details**: Include:
   - What you were trying to do
   - What happened instead
   - Error messages received
   - Browser and device information
   - Steps to reproduce the issue
