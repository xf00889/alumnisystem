# Direct Messaging

This guide explains how to use direct messaging to communicate with your connections in the NORSU Alumni System.

## Overview

Direct messaging allows you to have private, one-on-one conversations with alumni you're connected with. You can send text messages and file attachments, view conversation history, and maintain ongoing professional relationships.

## Who Can Use Direct Messaging

- **Required**: Authenticated alumni account
- **Access Level**: All registered alumni users
- **Prerequisites**:
  - Verified email address
  - At least one accepted connection
  - Active account status

## Accessing the Messaging System

### Method 1: Main Navigation

#### Step 1: Navigate to Messages

1. Log in to your NORSU Alumni account
2. Look for **"Messages"** in the main navigation menu
3. Click on **"Messages"**

**Expected Result**: The messaging page loads with your conversation list on the left and the main conversation area on the right.

![Messaging Page](../../screenshots/messaging-main-page.png)

### Method 2: From Alumni Profile

#### Step 1: Open a Connected Alumni's Profile

1. Navigate to the Alumni Directory
2. Find an alumni you're connected with
3. Click on their profile card or **"View Profile"** button

#### Step 2: Click Message Button

1. On the profile page, locate the **"Message"** button
2. Click the **"Message"** button

**Expected Result**: You're redirected to the messaging page with a conversation with that alumni opened or created.

![Profile Message Button](../../screenshots/profile-message-button.png)

### Method 3: From Alumni Directory

#### Step 1: Find Connected Alumni

1. Go to the Alumni Directory
2. Look for alumni cards that show **"Message"** button
3. This button only appears for alumni you're already connected with

#### Step 2: Click Message Button

1. Click the **"Message"** button on the alumni card
2. The messaging page opens with that conversation

**Expected Result**: The messaging interface opens with the selected conversation ready for messaging.

![Directory Message Button](../../screenshots/directory-message-button.png)

## Starting Conversations

### Understanding Conversation Creation

Conversations are automatically created when:
- You click "Message" on a connected alumni's profile
- You click "Message" on a connected alumni's directory card
- A connected alumni sends you a message first

**Important**: You can only start conversations with alumni you're already connected with. If you're not connected, you must send a connection request first.

### Starting a New Conversation

#### Step 1: Access Messaging Page

1. Navigate to the messaging page using any method above
2. The conversation list appears on the left sidebar

#### Step 2: Find the Alumni

**Option A: From Existing Conversations**
- Scroll through your conversation list
- Click on an existing conversation to continue it

**Option B: Start New Conversation**
- Navigate to the alumni's profile
- Click the **"Message"** button
- A new conversation is created automatically

**Expected Result**: The conversation opens in the main area, ready for you to send messages.

![New Conversation](../../screenshots/new-conversation.png)

### Conversation List Overview

The left sidebar shows all your conversations with:
- **Profile Photo**: The other person's profile picture
- **Name**: Full name of the other participant
- **Last Message**: Preview of the most recent message
- **Timestamp**: When the last message was sent
- **Unread Badge**: Number of unread messages (if any)

Conversations are sorted by most recent activity at the top.

![Conversation List](../../screenshots/conversation-list.png)

## Sending Messages

### Sending Text Messages

#### Step 1: Select a Conversation

1. From the conversation list on the left, click on a conversation
2. The conversation opens in the main area
3. You'll see the message history and input area at the bottom

**Expected Result**: The conversation loads with all previous messages displayed in chronological order (oldest at top, newest at bottom).

![Conversation View](../../screenshots/conversation-view.png)

#### Step 2: Type Your Message

1. Click in the message input field at the bottom
2. The placeholder text says "Type your message..."
3. Type your message content

**Tips**:
- Keep messages clear and concise
- Use proper grammar and spelling
- Be professional in tone
- Break long messages into paragraphs

#### Step 3: Send the Message

1. After typing your message, click the **"Send"** button
2. Or press **Enter** on your keyboard (if enabled)

**Expected Result**:
- Your message appears in the conversation immediately
- The message shows your profile photo and name
- The timestamp is displayed
- The message input field clears
- The conversation list updates with your message as the latest

![Message Sent](../../screenshots/message-sent.png)

### Message Display

Your messages appear on the right side of the conversation with:
- Your profile photo
- Your name
- Message content
- Timestamp (e.g., "2 minutes ago", "Yesterday at 3:45 PM")

The other person's messages appear on the left side with their information.

### Sending Attachments

#### Step 1: Select a File

1. In the message input area, look for the **attachment icon** or **"Attach File"** button
2. Click the attachment button
3. A file browser window opens

**Expected Result**: Your computer's file selection dialog appears.

![Attach File Button](../../screenshots/attach-file-button.png)

#### Step 2: Choose Your File

1. Navigate to the file you want to send
2. Select the file
3. Click **"Open"** or **"Choose"**

**Supported File Types**:
- Documents: PDF, DOC, DOCX, TXT, XLS, XLSX, PPT, PPTX
- Images: JPG, JPEG, PNG, GIF
- Archives: ZIP, RAR
- Other: Check with system administrator for complete list

**File Size Limits**:
- Maximum file size: Typically 10-25 MB (check system settings)
- Larger files may need to be compressed or shared via other means

**Expected Result**: The file name appears in the message input area, indicating it's ready to send.

![File Selected](../../screenshots/file-selected.png)

#### Step 3: Add Optional Message

1. You can type a message to accompany the file
2. Or send the file without additional text
3. The message is optional

#### Step 4: Send the Attachment

1. Click the **"Send"** button
2. The file uploads and sends

**Expected Result**:
- A progress indicator may appear during upload
- Once complete, the message appears with the file attachment
- The file shows as a downloadable link or preview
- The recipient can download the file

![Attachment Sent](../../screenshots/attachment-sent.png)

### Attachment Display

Attachments appear in the conversation as:
- **File Icon**: Indicates file type
- **File Name**: Original name of the file
- **File Size**: Size of the file
- **Download Button**: Click to download the file

Images may display as thumbnails that can be clicked to view full size.

## Viewing Conversation History

### Scrolling Through Messages

#### Viewing Recent Messages

1. When you open a conversation, the most recent messages appear at the bottom
2. Scroll up to see older messages
3. The conversation loads more messages as you scroll

**Expected Result**: Messages load smoothly as you scroll, showing the complete conversation history.

### Message Organization

Messages are organized chronologically:
- **Oldest messages**: At the top
- **Newest messages**: At the bottom
- **Date separators**: Show when messages were sent (e.g., "Today", "Yesterday", "March 15, 2024")
- **Time stamps**: Show exact time for each message

![Conversation History](../../screenshots/conversation-history.png)

### Finding Specific Messages

**Scroll Method**:
1. Scroll through the conversation manually
2. Look for date separators to navigate by date
3. Use browser's find function (Ctrl+F or Cmd+F) to search for text

**Tips for Finding Messages**:
- Remember approximate date of the message
- Look for distinctive words or phrases
- Check attachments if you're looking for a file
- Note: Some systems may have a search feature within conversations

### Message Timestamps

Each message shows when it was sent:
- **Recent messages**: "Just now", "2 minutes ago", "1 hour ago"
- **Today's messages**: "Today at 3:45 PM"
- **Yesterday's messages**: "Yesterday at 10:30 AM"
- **Older messages**: "March 15, 2024 at 2:15 PM"

### Read Status

Messages may show read status indicators:
- **Sent**: Message delivered to server
- **Delivered**: Message delivered to recipient
- **Read**: Recipient has viewed the message (if enabled)

The system updates read status when the recipient opens the conversation.

## Managing Conversations

### Switching Between Conversations

#### Step 1: View Conversation List

1. The left sidebar shows all your conversations
2. Conversations are sorted by most recent activity
3. Unread conversations may be highlighted or show a badge

#### Step 2: Select a Conversation

1. Click on any conversation in the list
2. The conversation loads in the main area
3. Previous conversation is replaced

**Expected Result**: The selected conversation opens, showing all messages and the input area.

![Switch Conversations](../../screenshots/switch-conversations.png)

### Unread Message Indicators

Conversations with unread messages show:
- **Badge**: Number of unread messages (e.g., "3")
- **Bold Text**: Conversation name may be bold
- **Highlight**: Background color may be different
- **Position**: Unread conversations may appear at the top

When you open a conversation:
- Unread messages are marked as read automatically
- The unread badge disappears
- The conversation styling returns to normal

![Unread Messages](../../screenshots/unread-messages.png)

### Conversation Information

At the top of each conversation, you'll see:
- **Participant Name**: Name of the person you're messaging
- **Profile Photo**: Their profile picture
- **Online Status**: May show if they're currently online (if enabled)
- **Profile Link**: Click name or photo to view their full profile

### Refreshing Conversations

To check for new messages:
- **Auto-refresh**: The system may automatically check for new messages
- **Manual refresh**: Refresh your browser page
- **Notification**: You may receive notifications for new messages

When someone sends you a message:
- **In-app notification**: Bell icon shows new notification
- **Email notification**: You may receive an email (based on your settings)
- **Browser notification**: Desktop notification may appear (if enabled)
- **Unread badge**: Conversation shows unread count

### Notification Settings

To manage message notifications:
1. Go to your account settings
2. Look for notification preferences
3. Choose which notifications you want to receive:
   - In-app notifications
   - Email notifications
   - Browser notifications

## Tips and Best Practices

### Effective Messaging

1. **Be Clear and Concise**
   - Get to the point quickly
   - Use short paragraphs
   - Break up long messages

2. **Be Professional**
   - Use proper grammar and spelling
   - Maintain professional tone
   - Avoid slang or abbreviations

3. **Be Responsive**
   - Reply to messages promptly
   - If you need time, acknowledge receipt
   - Set expectations for response time

4. **Be Respectful**
   - Respect others' time
   - Don't send too many messages at once
   - Be courteous and polite

### Message Content

**Good Practices**:
- Introduce yourself in first messages
- State your purpose clearly
- Ask specific questions
- Provide context when needed
- Thank people for their time

**Avoid**:
- Sending spam or promotional content
- Sharing inappropriate content
- Using all caps (seems like shouting)
- Sending too many follow-ups
- Sharing sensitive personal information

### File Sharing Best Practices

1. **Check File Size**: Ensure files aren't too large
2. **Use Appropriate Formats**: Send common file types
3. **Scan for Viruses**: Ensure files are safe
4. **Name Files Clearly**: Use descriptive file names
5. **Compress When Needed**: Zip large files or multiple files

### Building Relationships

- **Start with Introduction**: Explain who you are and why you're reaching out
- **Find Common Ground**: Reference shared experiences or interests
- **Be Genuine**: Be authentic in your communication
- **Follow Up**: Continue conversations over time
- **Offer Value**: Share helpful information or resources

## Troubleshooting

### Cannot Send Messages

**Problem**: Message won't send when you click Send button

**Possible Causes**:
- Not connected with the recipient
- Internet connection issues
- Account restrictions
- System maintenance

**Solutions**:
1. Verify you're connected with the recipient
2. Check your internet connection
3. Refresh the page and try again
4. Clear browser cache
5. Try a different browser
6. Log out and log back in
7. Contact support if issue persists

### Messages Not Appearing

**Problem**: Sent messages don't appear in conversation

**Solutions**:
1. Wait a few seconds for processing
2. Refresh the page
3. Check internet connection
4. Verify message was actually sent (check for error)
5. Try sending again if no error occurred

### Cannot Upload Attachments

**Problem**: File won't upload or attach to message

**Possible Causes**:
- File too large
- Unsupported file type
- Internet connection issues
- Browser issues

**Solutions**:
1. Check file size (compress if too large)
2. Verify file type is supported
3. Try a different file
4. Check internet connection
5. Try a different browser
6. Clear browser cache
7. Disable browser extensions temporarily

### Conversation Not Loading

**Problem**: Conversation won't open or load messages

**Solutions**:
1. Refresh the page
2. Check internet connection
3. Clear browser cache and cookies
4. Try a different browser
5. Log out and log back in
6. Check if the other user's account is active
7. Contact support if issue persists

### Cannot Find a Conversation

**Problem**: Can't locate a specific conversation

**Solutions**:
1. Scroll through entire conversation list
2. Check if you're actually connected with that person
3. Verify the conversation exists (may have been deleted)
4. Try accessing from the person's profile
5. Refresh the page

### Attachment Won't Download

**Problem**: Cannot download file attachment from message

**Solutions**:
1. Click the download link again
2. Try right-click and "Save As"
3. Check your browser's download settings
4. Ensure you have storage space
5. Try a different browser
6. Check if file still exists on server

## Privacy and Security

### Message Privacy

- **Private Conversations**: Only you and the recipient can see messages
- **No Public Access**: Messages are not visible to other users
- **Secure Storage**: Messages are stored securely
- **Connection Required**: Only connections can message you

### What Recipients See

When you send a message, the recipient sees:
- Your name and profile photo
- Message content
- Timestamp
- Any attachments
- Your online status (if enabled)

### Protecting Your Privacy

1. **Don't Share Sensitive Information**
   - Avoid sharing passwords
   - Be careful with personal details
   - Don't share financial information
   - Protect your contact details

2. **Be Cautious with Attachments**
   - Don't open suspicious files
   - Scan downloads for viruses
   - Verify sender before opening files

3. **Report Inappropriate Messages**
   - Don't respond to spam
   - Report harassment or abuse
   - Block users if necessary
   - Contact administrators

### Security Best Practices

- Use strong passwords for your account
- Log out on shared computers
- Don't share your account credentials
- Keep your browser updated
- Use secure internet connections
- Be wary of phishing attempts

## Limitations and Restrictions

### Who You Can Message

- **Connections Only**: You can only message alumni you're connected with
- **Active Accounts**: Both accounts must be active
- **No Self-Messaging**: You cannot message yourself
- **Blocked Users**: You cannot message blocked users

### Message Limits

- **No Daily Limit**: Send as many messages as needed
- **Character Limit**: Messages may have a maximum length (typically 5000-10000 characters)
- **Attachment Size**: Files limited to maximum size (typically 10-25 MB)
- **Attachment Types**: Only certain file types allowed

### Conversation Limits

- **No Limit**: No limit on number of conversations
- **No Expiration**: Conversations don't expire
- **History Retained**: Message history is preserved
- **Storage**: Subject to system storage limits
