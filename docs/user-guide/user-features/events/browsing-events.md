# Browsing Events

## Overview

The Events feature allows you to discover and browse alumni events, including reunions, networking sessions, workshops, and social gatherings. You can view upcoming events, explore past events, and use search and filtering tools to find events that interest you.

## Who Can Use This Feature

- **User Role**: All authenticated alumni users
- **Permissions Required**: Active user account
- **Prerequisites**: Must be logged in to the system

## How to Access

1. Log in to your alumni account
2. Click on **"Events"** in the main navigation menu
3. You will be directed to the Events List page

## Key Features

- View all upcoming and past events
- Search events by title, description, or location
- Filter events by status (published, draft, cancelled, completed)
- View event details including date, time, location, and RSVP counts
- See virtual event indicators and links
- View event capacity and current attendance
- Paginated event listings for easy browsing

## Step-by-Step Guide

### Task 1: Viewing the Events List

The Events List page displays all events you have access to, organized chronologically.

1. **Navigate to the Events page**
   - Click on **"Events"** in the main navigation menu
   - **Expected Result**: You will see the Events List page with a grid or list of events

2. **Review the event cards**
   - Each event card displays:
     - Event title
     - Start and end date/time
     - Location (or "Virtual Event" indicator)
     - Event status (Published, Draft, Cancelled, Completed)
     - Number of RSVPs
     - Event image (if available)
   - **Expected Result**: You can quickly scan multiple events at once

3. **Navigate through pages**
   - If there are more than 12 events, use the pagination controls at the bottom
   - Click page numbers or "Next"/"Previous" buttons to browse more events
   - **Expected Result**: The page loads the next set of events

### Task 2: Viewing Upcoming Events

Focus on events that are scheduled for the future.

1. **Access the Events List page**
   - Navigate to **Events** from the main menu
   - **Expected Result**: The events list is displayed

2. **View upcoming events section**
   - Look for the "Upcoming Events" section or sidebar
   - This displays the next 5 upcoming published events
   - Events are ordered by start date (soonest first)
   - **Expected Result**: You see a quick preview of upcoming events

3. **Identify upcoming events in the main list**
   - Events with start dates in the future are upcoming events
   - Check the date displayed on each event card
   - **Expected Result**: You can distinguish between upcoming and past events

### Task 3: Viewing Past Events

Review events that have already concluded.

1. **Navigate to the Events List**
   - Go to the **Events** page
   - **Expected Result**: The events list is displayed

2. **Identify past events**
   - Past events are those where the end date has passed
   - These may be marked with a "Completed" status
   - Past events appear in the list alongside upcoming events
   - **Expected Result**: You can see events that have already occurred

3. **Review past event details**
   - Click on a past event to view its details
   - You can see who attended and event outcomes
   - **Expected Result**: Access to historical event information

### Task 4: Filtering Events by Status

Filter events based on their publication status.

1. **Locate the filter controls**
   - On the Events List page, look for the filter section
   - Find the "Status" filter dropdown or buttons
   - **Expected Result**: Filter controls are visible

2. **Select a status filter**
   - Choose from available statuses:
     - **Published**: Events that are live and visible to all users
     - **Draft**: Events still being prepared (admin only)
     - **Cancelled**: Events that have been cancelled
     - **Completed**: Events that have finished
   - Click on your desired status
   - **Expected Result**: The event list updates to show only events with the selected status

3. **Clear the filter**
   - To view all events again, clear the status filter
   - Look for a "Clear" or "All" option
   - **Expected Result**: All events are displayed again

### Task 5: Searching for Events

Use the search function to find specific events.

1. **Locate the search box**
   - On the Events List page, find the search input field
   - It's typically located at the top of the page
   - **Expected Result**: Search box is visible and ready for input

2. **Enter your search query**
   - Type keywords related to the event you're looking for
   - You can search by:
     - Event title
     - Description content
     - Location name
   - **Expected Result**: Search terms are entered in the search box

3. **Execute the search**
   - Press **Enter** or click the **Search** button
   - **Expected Result**: The page refreshes and displays only events matching your search terms

4. **Review search results**
   - Events matching your query are displayed
   - Matching text may be highlighted in the results
   - **Expected Result**: You see a filtered list of relevant events

5. **Clear the search**
   - To return to viewing all events, clear the search box
   - Delete your search terms and press Enter, or click a "Clear" button
   - **Expected Result**: All events are displayed again

### Task 6: Viewing Event Statistics

Get an overview of event activity in the system.

1. **Access the Events List page**
   - Navigate to **Events** from the main menu
   - **Expected Result**: The events page loads

2. **Review the statistics panel**
   - Look for a statistics or overview section (usually at the top)
   - You may see:
     - Total number of events
     - Number of published events
     - Number of upcoming events
     - Number of past events
     - Total participants across all events
     - Number of virtual events
   - **Expected Result**: You have a quick overview of event activity

### Task 7: Identifying Virtual Events

Recognize and access virtual events.

1. **Browse the Events List**
   - Navigate to the **Events** page
   - **Expected Result**: Events are displayed

2. **Look for virtual event indicators**
   - Virtual events are marked with:
     - "Virtual Event" as the location
     - A virtual/online icon or badge
     - The "is_virtual" indicator
   - **Expected Result**: You can easily identify which events are virtual

3. **View virtual event details**
   - Click on a virtual event to see its details
   - Look for the virtual meeting link
   - **Expected Result**: You can access the virtual event link when available

## Tips and Best Practices

- **Check regularly**: New events are added frequently, so check the Events page regularly to stay updated
- **Use filters**: If you're looking for specific types of events, use the status filter to narrow down your options
- **Search effectively**: Use specific keywords when searching to get more relevant results
- **Plan ahead**: Review upcoming events to plan your schedule and RSVP early
- **Virtual events**: For virtual events, make sure to note the virtual link and test your connection before the event starts
- **Event capacity**: Check the max participants and current RSVP count to ensure there's space available

## Common Use Cases

### Use Case 1: Finding Networking Events
Search for "networking" or "career" in the search box to find professional development events. Filter by upcoming events to see what's available soon.

### Use Case 2: Reviewing Past Events
Browse past events to see what activities you missed or to review events you attended. This helps you stay connected with alumni activities.

### Use Case 3: Planning Your Schedule
Use the upcoming events section to see what's coming up in the next few weeks. RSVP to events that fit your schedule.

### Use Case 4: Finding Virtual Events
If you can't attend in-person events, filter or search for virtual events that you can join from anywhere.

## Troubleshooting

### Issue: No Events Are Displayed
**Symptoms**: The events list is empty or shows "No events found"
**Solution**: 
- Check if you have any active filters or search terms that might be limiting results
- Clear all filters and search terms to view all events
- If still empty, there may be no events created yet in the system

### Issue: Search Returns No Results
**Symptoms**: Your search query doesn't return any matching events
**Solution**:
- Try using different or more general keywords
- Check your spelling
- Clear the search and browse all events to see what's available
- Try searching by location instead of title

### Issue: Can't See Event Details
**Symptoms**: Clicking on an event doesn't show the full details
**Solution**:
- Ensure you're logged in to your account
- Check your internet connection
- Try refreshing the page
- If the issue persists, contact system administrators

### Issue: Pagination Not Working
**Symptoms**: Can't navigate to other pages of events
**Solution**:
- Refresh the page and try again
- Clear your browser cache
- Try using a different browser
- Contact support if the issue continues
