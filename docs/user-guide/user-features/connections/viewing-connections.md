# Viewing Connections

This guide explains how to access your connections list, view connection profiles, and remove connections when needed.

## Overview

Your Connections list is your professional network within the NORSU Alumni System. Here you can view all alumni you're connected with, access their profiles, message them, and manage your network.

## Who Can View Connections

- **Required**: Authenticated alumni account
- **Access Level**: All registered alumni users
- **Prerequisites**: 
  - Completed profile setup
  - At least one accepted connection (to see connections)

## Accessing Your Connections

### Step 1: Navigate to My Connections

**Method 1: Main Navigation**
1. Log in to your NORSU Alumni account
2. Click on **"Connections"** in the main navigation menu
3. Select **"My Connections"** from the dropdown

**Method 2: Direct Link**
1. Some pages include direct links to your connections
2. Click **"View Connections"** or **"My Network"** links
3. You'll be taken directly to your connections page

**Method 3: Profile Page**
1. Go to your profile page
2. Look for a connections count or link
3. Click to view your full connections list

**Expected Result**: The My Connections page loads, displaying all your accepted connections in a grid layout.

![My Connections Page](../../screenshots/my-connections-page.png)

## Understanding the Connections Page

### Page Layout

The My Connections page includes:

1. **Header Section**
   - Page title: "My Connections"
   - Connection count: Total number of connections
   - Search bar (if available)
   - Filter options (if available)

2. **Connections Grid**
   - Connection cards displayed in a responsive grid
   - 3-4 cards per row on desktop
   - 2 cards per row on tablet
   - 1 card per row on mobile

3. **Pagination**
   - Navigation controls at the bottom
   - Page numbers
   - Previous/Next buttons
   - Shows 12 connections per page by default

### Connection Card Information

Each connection card displays:
- **Profile Photo**: The connection's profile picture or default avatar
- **Full Name**: First and last name
- **Username**: System username
- **Current Position**: Job title (if provided)
- **Graduation Info**: Year and course
- **Location**: City and province (if provided)
- **Connection Date**: When you connected (may be shown)
- **Action Buttons**: View Profile, Message, Remove

![Connection Card Example](../../screenshots/connection-card.png)

## Viewing Your Connections List

### Default View

When you first access My Connections:
- All your accepted connections are displayed
- Cards are arranged in a grid layout
- Most recent connections may appear first
- 12 connections are shown per page

### Browsing Connections

#### Scrolling Through Connections

1. Scroll down to view connection cards
2. The layout automatically adjusts to your screen size
3. Cards maintain consistent sizing and spacing

#### Using Pagination

When you have more than 12 connections:

1. **View Next Page**:
   - Click the **">"** button or next page number
   - The page reloads with the next set of connections

2. **Jump to Specific Page**:
   - Click on any visible page number
   - The page loads that specific set of connections

3. **Return to First Page**:
   - Click the **"<<"** button
   - You'll return to page 1

4. **Go to Last Page**:
   - Click the **">>"** button
   - Jump to the last page of connections

![Connections Pagination](../../screenshots/connections-pagination.png)

### Searching Connections (If Available)

If a search feature is available:

#### Step 1: Locate the Search Bar

1. Find the search bar at the top of the connections page
2. It may be labeled "Search Connections" or "Find Connection"

#### Step 2: Enter Search Terms

1. Type a name, username, or keyword
2. Press Enter or click the search button
3. Results filter to match your search

**Expected Result**: The connections list updates to show only matching connections.

#### Step 3: Clear Search

1. Clear the search box
2. Click a "Clear" or "Reset" button if available
3. Or refresh the page to see all connections again

### Filtering Connections (If Available)

Some systems allow filtering connections:

**Filter Options May Include**:
- Graduation year
- College or campus
- Course/program
- Location
- Employment status
- Industry

**To Apply Filters**:
1. Locate filter controls on the page
2. Select your filter criteria
3. Click "Apply Filters"
4. View filtered results

## Viewing Connection Profiles

### Accessing a Connection's Profile

#### Step 1: Locate the Connection

1. Browse or search for the connection you want to view
2. Find their card in the connections grid

#### Step 2: Open the Profile

**Method 1: View Profile Button**
1. Click the **"View Profile"** button on the connection card
2. The full profile page opens

**Method 2: Click Name or Photo**
1. Click on the connection's name or profile photo
2. The full profile page opens

**Expected Result**: You'll see the complete profile of your connection, including:
- Personal information
- Education history
- Work experience
- Skills and achievements
- Documents (if shared)
- Contact information (if provided)

![Connection Profile View](../../screenshots/connection-profile.png)

### What You Can See

As a connected user, you can view:
- **Full Profile**: Complete profile information
- **Contact Details**: Email, phone (if shared)
- **Education**: All education entries
- **Experience**: Work history
- **Skills**: Listed skills and expertise
- **Achievements**: Awards and accomplishments
- **Documents**: Shared documents (if any)

### Profile Actions

From a connection's profile, you can:
- **Message**: Send a direct message
- **View Documents**: Access shared documents

## Removing Connections

### When to Remove a Connection

Consider removing a connection if:
- You no longer want them in your network
- The connection is no longer relevant
- You want to clean up your connections list
- The account appears inactive or suspicious
- You connected by mistake

**Important**: Removing a connection is permanent. You'll need to send a new connection request to reconnect.

### How to Remove a Connection

#### Method 1: From Connection Card

##### Step 1: Locate the Remove Option

1. Find the connection you want to remove
2. Look for a **"Remove"** button or icon on the card
3. It may be in a dropdown menu or visible as a button

##### Step 2: Click Remove

1. Click the **"Remove"** or **"Remove Connection"** button
2. A confirmation dialog may appear

##### Step 3: Confirm Removal

1. Read the confirmation message
2. Click **"Confirm"**, **"Yes"**, or **"Remove"** to proceed
3. Or click **"Cancel"** to keep the connection

**Expected Result**:
- A success message appears: "Connection with [Name] has been removed"
- The connection card is removed from your list
- The connection count decreases by one
- You can no longer message this user
- They can no longer see your full profile

![Remove Connection Confirmation](../../screenshots/remove-connection.png)

#### Method 2: From Profile Page

##### Step 1: Open the Profile

1. Navigate to the connection's profile page
2. Locate the connection actions section

##### Step 2: Find Remove Option

1. Look for a **"Remove Connection"** button or link
2. It may be in a dropdown menu or settings area

##### Step 3: Remove and Confirm

1. Click **"Remove Connection"**
2. Confirm the removal in the dialog
3. You'll be redirected back to your connections list

### After Removing a Connection

**What Happens**:
- The connection is immediately removed
- You both lose access to each other's full profiles
- You can no longer message each other
- Previous messages may remain visible (depending on system settings)
- The other user may receive a notification (depending on settings)

**To Reconnect**:
- You'll need to send a new connection request
- They'll need to accept it again
- Previous conversation history may or may not be restored

## Troubleshooting

### Connections Not Displaying

**Problem**: My Connections page is empty or not loading

**Solutions**:
1. Refresh the page
2. Check your internet connection
3. Verify you have accepted connections
4. Clear browser cache
5. Try a different browser
6. Log out and log back in

### Cannot View Connection Profile

**Problem**: Profile page won't load or shows error

**Solutions**:
1. Refresh the page
2. Check if the connection's account is still active
3. Verify you're still connected (they may have removed you)
4. Try accessing from the connections list again
5. Contact support if issue persists

### Message Button Not Working

**Problem**: Cannot message a connection

**Solutions**:
1. Verify you're still connected
2. Check if messaging is enabled
3. Ensure JavaScript is enabled
4. Try a different browser
5. Check your internet connection
6. Contact support if needed

### Remove Connection Failed

**Problem**: Cannot remove a connection

**Solutions**:
1. Refresh the page and try again
2. Check your internet connection
3. Try from the profile page instead
4. Clear browser cache
5. Contact support if issue persists

### Connection Count Incorrect

**Problem**: Connection count doesn't match visible connections

**Solutions**:
1. Refresh the page
2. Check all pages (use pagination)
3. Clear filters or search
4. Log out and log back in
5. Report the issue to support
