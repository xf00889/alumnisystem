# Mentee Requests

This guide explains how to manage your mentorship requests, track their status, and use your mentee dashboard effectively.

## Overview

As a mentee, you can request mentorship from experienced alumni, track the status of your requests, and manage your active mentorships. The mentee dashboard provides a centralized view of all your mentorship activities, including pending requests, active mentorships, upcoming meetings, and progress updates.

## Who Can Use This Feature

- **Required**: Authenticated alumni account
- **Access Level**: All registered alumni users (mentees)
- **Prerequisites**: 
  - Completed profile setup
  - Verified email address
  - Active account status

## How to Access

### Access Mentee Dashboard

1. Log in to your NORSU Alumni account
2. Click on **"Mentorship"** in the main navigation menu
3. Select **"My Dashboard"** or **"Mentee Dashboard"**

**Alternative Access**:
- From your main dashboard, click on the mentorship widget
- Navigate to **Mentorship > Mentee Dashboard**
- Click on notification links related to mentorship

**Expected Result**: Your mentee dashboard loads, showing all your mentorship activities.

![Mentee Dashboard](../../screenshots/mentorship/mentee-dashboard.png)

## Key Features

- **View Pending Requests**: See all mentorship requests awaiting mentor response
- **Track Active Mentorships**: Monitor your ongoing mentorship relationships
- **View Completed Mentorships**: Review past mentorship experiences
- **See Rejected Requests**: Understand which requests were declined
- **Upcoming Meetings**: View scheduled meetings with mentors
- **Recent Messages**: Access latest communications
- **Progress Updates**: Track your mentorship progress
- **Quick Actions**: Send messages, schedule meetings, update progress

## Step-by-Step Guide

### Task 1: Understanding Your Mentee Dashboard

#### Step 1: Dashboard Overview

When you access your mentee dashboard, you'll see several sections:

**Statistics Cards** (Top of page):
- **Active Mentorships**: Number of current mentorship relationships
- **Pending Requests**: Number of requests awaiting response
- **Completed Mentorships**: Total mentorships you've completed
- **Upcoming Meetings**: Number of scheduled meetings

**Main Sections**:
1. **Pending Requests**: Requests sent and awaiting mentor approval
2. **Active Mentorships**: Current mentorship relationships
3. **Completed Mentorships**: Finished mentorship programs
4. **Rejected Requests**: Declined mentorship requests
5. **Upcoming Meetings**: Scheduled meetings with mentors
6. **Recent Messages**: Latest mentorship communications
7. **Progress Updates**: Recent progress entries

**Expected Result**: You have a complete overview of your mentorship journey.

![Dashboard Sections](../../screenshots/mentorship/dashboard-sections.png)

### Task 2: Viewing and Managing Pending Requests

#### Step 1: Access Pending Requests Section

1. Scroll to the **"Pending Requests"** section on your dashboard
2. This section shows all mentorship requests you've sent that are awaiting mentor response

**Expected Result**: You see a list of pending requests with mentor information.

![Pending Requests](../../screenshots/mentorship/pending-requests.png)

#### Step 2: Review Request Details

Each pending request card displays:

- **Mentor Information**:
  - Name and profile photo
  - Current position and company
  - Expertise areas

- **Request Information**:
  - Date request was sent
  - Skills you're seeking
  - Your career goals
  - Your message to the mentor

- **Status**: "Pending" badge indicating awaiting response

- **Actions**:
  - **View Mentor Profile**: Review mentor's full profile
  - **Cancel Request** (if available): Withdraw your request

**Expected Result**: You can review all details of your pending requests.

#### Step 3: Cancel a Pending Request (Optional)

If you want to withdraw a request:

1. Locate the request you want to cancel
2. Click the **"Cancel Request"** button
3. Confirm the cancellation in the popup dialog

**Expected Result**: 
- Request is removed from pending list
- Mentor is notified of cancellation
- You can send a new request to the same mentor later

**When to Cancel**:
- You found a better mentor match
- Your goals have changed
- You're no longer seeking mentorship
- The mentor hasn't responded for a long time

### Task 3: Managing Active Mentorships

#### Step 1: View Active Mentorships

1. Scroll to the **"Active Mentorships"** section
2. This shows all approved mentorship relationships

**Expected Result**: You see cards for each active mentorship.

![Active Mentorships](../../screenshots/mentorship/active-mentorships.png)

#### Step 2: Review Mentorship Details

Each active mentorship card includes:

**Mentor Information**:
- Name and profile photo
- Current position
- Expertise areas

**Mentorship Details**:
- Start date
- Duration (if set)
- Expected end date (if set)
- Progress percentage
- Current status (Active, Paused, etc.)

**Timeline Information** (if set):
- Milestones and their status
- Completed objectives
- Upcoming goals

**Quick Actions**:
- **Send Message**: Open messaging interface
- **Schedule Meeting**: Create a new meeting
- **View Progress**: See detailed progress
- **Update Progress**: Add progress notes

**Expected Result**: You have complete visibility into each mentorship relationship.

#### Step 3: Send a Message to Your Mentor

1. On the mentorship card, click **"Send Message"**
2. A message modal or page opens
3. Type your message in the text area
4. Optionally attach files (if supported)
5. Click **"Send"**

**Expected Result**: 
- Message is sent to your mentor
- You're redirected to the messaging interface
- Mentor receives a notification

![Send Message](../../screenshots/mentorship/send-message.png)

**Message Tips**:
- Be clear and concise
- Ask specific questions
- Share updates on your progress
- Schedule meetings through messages
- Be professional and respectful

#### Step 4: Schedule a Meeting

1. Click **"Schedule Meeting"** on the mentorship card
2. Fill in the meeting details:
   - **Title**: Meeting purpose (e.g., "Monthly Check-in")
   - **Description**: Meeting agenda or topics
   - **Date and Time**: When the meeting will occur
   - **Duration**: Length in minutes (default: 60)
   - **Meeting Link**: Video call URL or location

3. Click **"Schedule"** or **"Create Meeting"**

**Expected Result**: 
- Meeting is created and appears in upcoming meetings
- Mentor receives a notification
- Both parties can see the meeting details

![Schedule Meeting](../../screenshots/mentorship/schedule-meeting.png)

**Meeting Best Practices**:
- Schedule at least 3-5 days in advance
- Provide clear agenda
- Include video call link
- Set realistic duration
- Send reminder message before meeting

#### Step 5: Update Progress

1. Click **"Update Progress"** or **"Add Progress Note"**
2. A progress update form appears
3. Fill in:
   - **Title**: Brief description (e.g., "Completed Python Course")
   - **Description**: Detailed progress update
   - **Completed Items**: What you've accomplished
   - **Next Steps**: What you plan to do next

4. Optionally update progress percentage
5. Click **"Save"** or **"Submit"**

**Expected Result**: 
- Progress update is saved
- Mentor can see your update
- Progress percentage updates (if changed)
- Update appears in progress history

![Update Progress](../../screenshots/mentorship/update-progress.png)

**What to Include in Progress Updates**:
- Skills learned or improved
- Projects completed
- Challenges overcome
- Goals achieved
- Questions or concerns
- Next objectives

### Task 4: Viewing Request Status

#### Understanding Request Statuses

Your mentorship requests can have several statuses:

1. **Pending** (Yellow/Warning Badge)
   - Request sent, awaiting mentor response
   - No action taken yet
   - Mentor is reviewing your request

2. **Approved** (Green/Success Badge)
   - Mentor accepted your request
   - Mentorship is now active
   - Moves to "Active Mentorships" section

3. **Rejected** (Red/Danger Badge)
   - Mentor declined your request
   - Moves to "Rejected Requests" section
   - You can request from other mentors

4. **Completed** (Blue/Info Badge)
   - Mentorship has ended successfully
   - Moves to "Completed Mentorships" section
   - Goals were achieved

5. **Cancelled** (Gray/Secondary Badge)
   - You or mentor cancelled the mentorship
   - Relationship ended early
   - No longer active

**Expected Result**: You can quickly identify the status of each request.

![Request Statuses](../../screenshots/mentorship/request-statuses.png)

#### Step 1: Check Request Status

1. Navigate to your mentee dashboard
2. Look at the section where your request appears:
   - **Pending Requests**: Still waiting
   - **Active Mentorships**: Approved and ongoing
   - **Rejected Requests**: Declined by mentor
   - **Completed Mentorships**: Successfully finished

**Expected Result**: You know exactly where each request stands.

#### Step 2: Respond to Status Changes

**If Approved**:
1. You receive a notification
2. Send a thank you message to your mentor
3. Discuss goals and expectations
4. Schedule your first meeting
5. Set up a timeline (if required)

**If Rejected**:
1. Review the rejection (mentor may provide reason)
2. Don't take it personally
3. Search for other mentors
4. Refine your request approach
5. Send requests to alternative mentors

**Expected Result**: You take appropriate action based on status.

#### Step 1: Access Completed Mentorships

Each completed mentorship card shows:

### Task 6: Managing Meetings

#### Step 1: View Upcoming Meetings

1. Locate the **"Upcoming Meetings"** section on your dashboard
2. This shows all scheduled meetings with your mentors

**Expected Result**: You see a list of upcoming meetings sorted by date.

![Upcoming Meetings](../../screenshots/mentorship/upcoming-meetings.png)

#### Step 2: Review Meeting Details

Each meeting card displays:

- **Meeting Title**: Purpose of the meeting
- **Mentor Name**: Who you're meeting with
- **Date and Time**: When the meeting occurs
- **Duration**: Length of the meeting
- **Meeting Link**: Video call or location URL
- **Description**: Meeting agenda or notes
- **Status**: Scheduled, Completed, Cancelled, or Rescheduled

**Actions**:
- **Join Meeting**: Click link to join (when time arrives)
- **View Details**: See full meeting information
- **Add to Calendar**: Export to your calendar
- **Cancel** (if allowed): Cancel the meeting

**Expected Result**: You're prepared for all upcoming meetings.

#### Step 3: Join a Meeting

1. When meeting time approaches, go to your dashboard
2. Click the **"Join Meeting"** button or meeting link
3. You'll be directed to the video call or location

**Expected Result**: You can easily access your scheduled meetings.

**Meeting Preparation**:
- Review meeting agenda beforehand
- Prepare questions or topics
- Have materials ready
- Test video/audio if virtual
- Be on time

### Task 7: Tracking Progress

#### Step 1: View Progress Updates

1. Scroll to the **"Progress Updates"** section
2. This shows recent progress entries from you and your mentor

**Expected Result**: You see a timeline of progress updates.

![Progress Updates](../../screenshots/mentorship/progress-updates.png)

#### Step 2: Review Progress History

Each progress update shows:

- **Title**: Brief description
- **Description**: Detailed update
- **Created By**: Who added the update (you or mentor)
- **Date**: When it was added
- **Mentorship**: Which mentorship it relates to

#### Step 3: Add Quick Progress Update

1. Find the **"Quick Progress Update"** section or button
2. Enter a brief note about your progress
3. Optionally update progress percentage

**Expected Result**: 
- Quick update is saved
- Mentor is notified
- Progress percentage updates

![Quick Progress](../../screenshots/mentorship/quick-progress.png)

**Quick Update Examples**:
- "Completed Module 3 of the online course"
- "Applied for 5 jobs this week"
- "Finished the project prototype"
- "Had a great networking event"

### Task 8: Setting and Tracking Timeline

#### Step 1: Set Mentorship Timeline (If Required)

Some mentorships require setting a timeline with milestones:

1. On your active mentorship card, click **"Set Timeline"**
2. Fill in the timeline form:
   - **Expected End Date**: When mentorship should complete
   - **Milestones**: List of goals with timeframes
   - **Notes**: Additional timeline information

3. Add milestones:
   - **Period**: Week/Day range (e.g., "1-2", "3-4")
   - **Description**: What should be achieved
   - Click **"Add Milestone"** to add more

4. Click **"Save Timeline"**

**Expected Result**: 
- Timeline is set for the mentorship
- Milestones appear on dashboard
- Both you and mentor can track progress

![Set Timeline](../../screenshots/mentorship/set-timeline.png)

#### Step 2: Update Milestone Status

1. View your mentorship timeline
2. For each milestone, you can update status:
   - **Not Started**: Haven't begun yet
   - **In Progress**: Currently working on it
   - **Completed**: Finished successfully

3. Click on a milestone to change its status
4. Or use the **"Update Milestones"** button

**Expected Result**: 
- Milestone status updates
- Progress percentage recalculates
- Visual progress bar updates

**Timeline Tips**:
- Set realistic milestones
- Break large goals into smaller ones
- Update status regularly
- Discuss timeline with mentor
- Adjust as needed

## Common Use Cases

### Use Case 1: First-Time Mentee
**Scenario**: You've never had a mentor before

**Steps**:
1. Send requests to 2-3 potential mentors
2. Wait for responses (check dashboard daily)
3. Once approved, send introductory message
4. Schedule first meeting to discuss goals
5. Set up timeline with mentor's guidance
6. Update progress regularly

### Use Case 2: Multiple Mentorships
**Scenario**: You have mentors for different areas

**Steps**:
1. Use dashboard to manage all mentorships
2. Keep separate progress notes for each
3. Schedule meetings strategically
4. Update each mentorship independently
5. Track different goals and timelines

### Use Case 3: Career Transition
**Scenario**: Using mentorship to change careers

**Steps**:
1. Request mentor in target industry
2. Set clear transition goals in timeline
3. Schedule frequent check-ins
4. Update progress on transition activities
5. Share job applications and interviews
6. Seek feedback on career materials

### Use Case 4: Skill Development
**Scenario**: Learning specific technical skills

**Steps**:
1. Request mentor with expertise in skill
2. Set learning milestones in timeline
3. Share completed courses or projects
4. Schedule meetings to review work
5. Update progress with skill improvements
6. Track proficiency growth

## Tips and Best Practices

### Effective Mentee Practices

1. **Be Proactive**
   - Don't wait for mentor to reach out
   - Schedule meetings regularly
   - Update progress frequently
   - Ask questions actively

2. **Set Clear Goals**
   - Define what you want to achieve
   - Break goals into milestones
   - Track progress systematically
   - Adjust goals as needed

3. **Communicate Regularly**
   - Send updates between meetings
   - Respond promptly to messages
   - Share both successes and challenges
   - Be honest about progress

4. **Come Prepared**
   - Prepare for meetings in advance
   - Have specific questions ready
   - Share work or materials
   - Take notes during meetings

5. **Show Appreciation**
   - Thank your mentor regularly
   - Acknowledge their time and effort
   - Share how their advice helped
   - Provide feedback on mentorship

6. **Take Action**
   - Implement mentor's advice
   - Complete agreed-upon tasks
   - Follow through on commitments
   - Show progress between meetings

### Dashboard Management Tips

1. **Check Daily**: Review dashboard for updates and notifications
2. **Update Regularly**: Add progress notes at least weekly
3. **Respond Quickly**: Reply to mentor messages promptly
4. **Track Meetings**: Keep meeting notes and action items
5. **Monitor Progress**: Review progress percentage and milestones
6. **Stay Organized**: Use dashboard sections to stay on top of activities

### Request Management Tips

1. **Be Patient**: Mentors may take 3-7 days to respond
2. **Follow Up**: If no response after 2 weeks, send a polite message
3. **Multiple Requests**: Send to 2-3 mentors to increase chances
4. **Quality Over Quantity**: Focus on good matches, not many requests
5. **Learn from Rejections**: Use feedback to improve future requests

## Troubleshooting

### Cannot Access Dashboard

**Problem**: Dashboard page won't load

**Solutions**:
1. Refresh the page
2. Clear browser cache
3. Check internet connection
4. Try a different browser
5. Log out and log back in
6. Contact support if issue persists

### Request Status Not Updating

**Problem**: Status shows "Pending" but mentor responded

**Solutions**:
1. Refresh the dashboard
2. Check your notifications
3. Log out and log back in
4. Wait a few minutes for sync
5. Contact support if still not updated

### Cannot Send Message

**Problem**: Message button doesn't work

**Solutions**:
1. Ensure mentorship is active (approved)
2. Check if mentor is still active
3. Refresh the page
4. Try from messaging page directly
5. Check browser console for errors

### Meeting Not Showing

**Problem**: Scheduled meeting doesn't appear

**Solutions**:
1. Refresh the dashboard
2. Check if meeting was cancelled
3. Verify meeting date is in future
4. Look in past meetings if date passed
5. Contact mentor to confirm

### Progress Not Saving

**Problem**: Progress updates don't save

**Solutions**:
1. Check all required fields are filled
2. Ensure description is not too long
3. Check internet connection
4. Try again after refreshing
5. Use quick progress update instead

### Timeline Not Displaying

**Problem**: Timeline milestones don't show

**Solutions**:
1. Verify timeline was set
2. Refresh the page
3. Check if mentorship is active
4. Ask mentor if they set timeline
5. Try setting timeline again

## Privacy and Security

### Your Information

- **Dashboard Privacy**: Only you can see your full dashboard
- **Mentor Access**: Mentors see only their mentorships with you
- **Progress Visibility**: Only you and your mentor see progress updates
- **Message Privacy**: Messages are private between you and mentor
- **Meeting Privacy**: Only participants see meeting details

### Data Security

- All communications are encrypted
- Progress data is securely stored
- Meeting links are private
- File attachments are scanned
- Account activity is logged
