# Feedback

## Overview

The feedback system allows you to communicate directly with system administrators by sharing your thoughts, reporting issues, requesting new features, or providing general comments about the NORSU Alumni System. Your input helps improve the platform for the entire alumni community.

## Who Can Use This Feature

- **User Role**: All authenticated users (registered alumni)
- **Permissions Required**: Active user account with verified email
- **Prerequisites**: Must be logged in to submit or view feedback

## Key Capabilities

- **Submit Feedback**: Share bug reports, feature requests, or general comments
- **Track Submissions**: View all your submitted feedback and their current status
- **File Attachments**: Include screenshots or documents to support your feedback
- **Priority Levels**: Indicate the urgency of your feedback
- **Status Updates**: Monitor how administrators are handling your feedback

## Available Features

- [Submitting Feedback](submitting-feedback.md) - Complete guide to submitting and tracking feedback

## Quick Start

1. Navigate to your profile menu and select "Submit Feedback"
2. Choose the appropriate category for your feedback
3. Enter a descriptive subject and detailed message
4. Set the priority level
5. Optionally attach supporting files
6. Submit your feedback and track it in "My Feedbacks"

## Feedback Categories

- **General**: General comments or suggestions about the system
- **Bug Report**: Report technical issues, errors, or unexpected behavior
- **Feature Request**: Suggest new features or improvements
- **Content**: Provide feedback about content quality or accuracy
- **Other**: Feedback that doesn't fit into other categories

## Feedback Status

- **Pending**: Awaiting administrator review
- **In Progress**: Being actively worked on by administrators
- **Resolved**: Issue has been addressed or request has been fulfilled
- **Closed**: Feedback has been reviewed and closed

## Tips for Effective Feedback

- Be specific and provide detailed information
- Include screenshots for visual issues
- Use appropriate categories and priority levels
- Submit one topic per feedback
- Check back for administrator responses
