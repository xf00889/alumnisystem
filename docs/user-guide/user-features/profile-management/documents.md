# Documents

## Overview

The Document Uploads feature allows you to upload and manage important academic and professional documents such as transcripts, certificates, diplomas, and resumes. These documents can be used for verification purposes, job applications, and showcasing your credentials to other alumni and potential employers.

## Who Can Use This Feature

- **All authenticated users** with verified email addresses
- **Alumni who want to store and share credentials**
- **Users applying for jobs through the system**

## Prerequisites

- Active account with verified email
- Completed registration process
- Logged into the system
- Documents in supported formats (PDF, DOC, DOCX, JPG, PNG)

## How to Access

1. Log into your NORSU Alumni System account
2. Navigate to your profile page
3. Click the "Edit Profile" button
4. Scroll to the "Documents" section
5. Or visit: `/accounts/profile/update/` and navigate to the Documents tab

## Key Features

- **Multiple Document Types**: Upload transcripts, certificates, diplomas, resumes, and other documents
- **Secure Storage**: Documents are stored securely and privately
- **File Format Support**: Supports PDF, Word documents, and images
- **Document Management**: View, download, and delete uploaded documents
- **Verification Status**: Track which documents have been verified by administrators
- **Job Application Integration**: Use uploaded documents for job applications
- **Privacy Control**: Documents are private by default

## Step-by-Step Guide

### Task 1: Uploading a Transcript

1. **Navigate to the Documents section**
   - Go to your profile edit page
   - Find the "Documents" section
   - **Expected Result**: Documents section is displayed

2. **Locate the Transcript upload area**
   - Find the "Academic Transcript" or "Transcript" section
   - **Expected Result**: Upload form for transcript is visible


3. **Enter a document title**
   - Type a descriptive title (e.g., "Official Transcript of Records 2020")
   - This helps you identify the document later
   - **Expected Result**: Title is entered

4. **Click "Choose File" or "Browse"**
   - Click the file selection button
   - **Expected Result**: File browser opens

5. **Select your transcript file**
   - Navigate to the file location on your computer
   - Select the transcript file (PDF recommended)
   - **Expected Result**: File is selected, filename appears

6. **Verify file details**
   - Check that the correct file is selected
   - Verify file size is within limits (typically 5MB max)
   - **Expected Result**: File details are displayed

7. **Click "Upload" or "Save"**
   - Click the upload button
   - Wait for upload to complete
   - **Expected Result**: Success message appears, document is uploaded

### Task 2: Uploading a Certificate

1. **Navigate to the Certificates section**
   - Find the "Certificates" upload area
   - **Expected Result**: Certificate upload form is visible

2. **Enter certificate title**
   - Type the certificate name (e.g., "AWS Certified Solutions Architect")
   - **Expected Result**: Title is entered

3. **Select certificate file**
   - Click "Choose File"
   - Select your certificate file
   - **Expected Result**: File is selected

4. **Upload the certificate**
   - Click "Upload" or "Save"
   - **Expected Result**: Certificate is uploaded successfully

### Task 3: Uploading a Diploma

1. **Navigate to the Diploma section**
   - Find the "Diploma" upload area
   - **Expected Result**: Diploma upload form is visible

2. **Enter diploma title**
   - Type a descriptive title (e.g., "Bachelor of Science in Information Technology Diploma")
   - **Expected Result**: Title is entered

3. **Select diploma file**
   - Click "Choose File"
   - Select your diploma file (scan or photo)
   - **Expected Result**: File is selected

4. **Upload the diploma**
   - Click "Upload" or "Save"
   - **Expected Result**: Diploma is uploaded successfully

### Task 4: Uploading a Resume

1. **Navigate to the Resume section**
   - Find the "Resume/CV" upload area
   - **Expected Result**: Resume upload form is visible

2. **Enter resume title**
   - Type a title (e.g., "Current Resume 2025")
   - **Expected Result**: Title is entered

3. **Select resume file**
   - Click "Choose File"
   - Select your resume file (PDF or Word document)
   - **Expected Result**: File is selected

4. **Upload the resume**
   - Click "Upload" or "Save"
   - **Expected Result**: Resume is uploaded successfully


### Task 5: Viewing Uploaded Documents

1. **Navigate to your profile page**
   - Go to your profile (not edit mode)
   - **Expected Result**: Profile page is displayed

2. **Scroll to the Documents section**
   - Find the "Documents" or "Uploaded Documents" section
   - **Expected Result**: List of uploaded documents is visible

3. **View document details**
   - See document title, type, and upload date
   - Check verification status (verified/unverified)
   - **Expected Result**: Document information is displayed

4. **Download a document**
   - Click the document title or download icon
   - **Expected Result**: Document downloads to your computer

### Task 6: Deleting a Document

1. **Navigate to your profile page**
   - Go to your profile
   - **Expected Result**: Profile page is displayed

2. **Locate the document to delete**
   - Scroll to the Documents section
   - Find the document you want to remove
   - **Expected Result**: Document is visible in the list

3. **Click the "Delete" button**
   - Look for the delete icon (trash can) or "Delete" button
   - **Expected Result**: Confirmation dialog appears

4. **Confirm deletion**
   - Read the confirmation message
   - Click "Yes" or "Confirm" to proceed
   - Click "No" or "Cancel" to abort
   - **Expected Result**: Document is deleted if confirmed

5. **Verify deletion**
   - Check that the document no longer appears in your list
   - **Expected Result**: Document is removed from profile

**Warning**: Deleted documents cannot be recovered. You'll need to upload them again if needed.

## Troubleshooting

### Issue: File upload fails
**Symptoms**: Upload button doesn't work or error message appears

**Solution**:
- Check file size (must be under 5MB)
- Verify file format is supported (PDF, DOC, DOCX, JPG, PNG)
- Check internet connection
- Try a different browser
- Compress large files before uploading
- Convert to PDF if using unsupported format
- Contact support if issue persists


### Issue: File too large error
**Symptoms**: "File size exceeds maximum" error

**Solution**:
- Check file size (typically 5MB maximum)
- Compress PDF files using online tools
- Reduce image resolution for scanned documents
- Split large documents into multiple files
- Convert images to PDF and compress
- Contact support if you need to upload larger files

### Issue: Unsupported file format
**Symptoms**: "Invalid file type" or "Unsupported format" error

**Solution**:
- Convert file to PDF (recommended)
- Use supported formats: PDF, DOC, DOCX, JPG, PNG
- Check file extension is correct
- Don't use ZIP or RAR files
- Convert images to JPG or PNG
- Contact support for help with file conversion

### Issue: Document not appearing after upload
**Symptoms**: Upload succeeds but document doesn't show in list

**Solution**:
- Refresh the page
- Wait a few seconds for processing
- Check internet connection during upload
- Verify upload completed successfully
- Try uploading again
- Contact support if document is missing

### Issue: Cannot download document
**Symptoms**: Download button doesn't work or file won't open

**Solution**:
- Check internet connection
- Try a different browser
- Clear browser cache
- Ensure you have PDF reader installed
- Try right-click and "Save As"
- Contact support if download consistently fails

### Issue: Cannot delete document
**Symptoms**: Delete button doesn't work or document reappears

**Solution**:
- Ensure you're logged in
- Check that you own the document
- Try refreshing the page
- Clear browser cache
- Try a different browser
- Contact support if deletion fails

### Issue: Document not verified
**Symptoms**: Document shows as "Unverified" for extended period

**Solution**:
- Verification is done manually by administrators
- Allow 3-5 business days for verification
- Ensure document is clear and readable
- Check that you uploaded the correct document type
- Contact support if verification is urgent
- Reupload if document quality is poor

## File Requirements

### File Formats
**Supported**:
- PDF (.pdf) - Recommended
- Microsoft Word (.doc, .docx)
- JPEG Image (.jpg, .jpeg)
- PNG Image (.png)

**Not Supported**:
- Compressed files (.zip, .rar, .7z)
- Executable files (.exe)
- Video files (.mp4, .avi)
- Audio files (.mp3, .wav)

### File Size Limits
- Maximum file size: 5MB per document
- Recommended size: Under 2MB for faster uploads
- Large files may take longer to upload
- Compress files if they exceed the limit

### Document Quality
- Scanned documents: 300 DPI minimum
- Images: Clear and readable text
- PDF: Text should be selectable (not just image)
- No watermarks or obstructions
- Full document visible (no cropping)


## Document Verification

### What is Verification?
- Administrators review uploaded documents for authenticity
- Verified documents have a "Verified" badge
- Verification adds credibility to your profile
- Some features may require verified documents

### Verification Process
1. Upload your document
2. Administrator reviews the document
3. Document is marked as verified or flagged for issues
4. You receive notification of verification status
5. Verified badge appears on your profile

### Verification Timeline
- Typical verification: 3-5 business days
- Complex documents may take longer
- Urgent requests can be expedited (contact support)
- You'll be notified when verification is complete

### What Gets Verified
- Authenticity of the document
- Clarity and readability
- Correct document type
- Completeness of information
- Official seals or signatures (if applicable)

## Privacy and Security

### Document Privacy
- All uploaded documents are private by default
- Only you and administrators can view your documents
- Documents are not shared publicly
- You control who can access your documents

### Data Security
- Documents are encrypted during upload
- Stored securely on protected servers
- Access is logged for security
- Regular security audits performed
- Compliant with data protection regulations

### Document Sharing
- You can choose to share documents with:
  - Potential employers (during job applications)
  - Mentors (if relevant to mentorship)
  - Administrators (for verification)
- Sharing is always with your explicit consent
- You can revoke access at any time

### Required Documents

### Document Selection
- Choose relevant documents for each application
