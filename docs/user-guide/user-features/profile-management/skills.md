# Skills

## Overview

The Skills Management feature allows you to document and showcase your professional and technical skills. You can add multiple skills, categorize them by type, indicate proficiency levels, and track years of experience. Your skills are used for job matching, mentorship pairing, and helping other alumni find experts in specific areas.

## Who Can Use This Feature

- **All authenticated users** with verified email addresses
- **Alumni who want to showcase their expertise**
- **Users seeking job opportunities and mentorship**

## Prerequisites

- Active account with verified email
- Completed registration process
- Logged into the system

## How to Access

1. Log into your NORSU Alumni System account
2. Navigate to your profile page
3. Click the "Edit Profile" button
4. Scroll to the "Skills" section
5. Or visit: `/accounts/profile/update/` and navigate to the Skills tab

## Key Features

- **Multiple Skills**: Add unlimited skills to your profile
- **Skill Categories**: Organize skills by type (Technical, Soft Skills, Languages, etc.)
- **Proficiency Levels**: Rate your expertise from Beginner to Expert
- **Experience Tracking**: Record years of experience with each skill
- **Primary Skills**: Mark your key skills
- **Last Used Date**: Track when you last used each skill
- **Job Matching**: Skills are used to match you with relevant job opportunities
- **Searchable**: Other alumni can find you based on your skills
- **Edit and Delete**: Modify or remove skills as needed

## Step-by-Step Guide

### Task 1: Adding a New Skill

1. **Navigate to the Skills section**
   - Go to your profile edit page
   - Find the "Skills" section
   - **Expected Result**: Skills section is displayed with existing skills

2. **Click "Add Skill" button**
   - Look for the "Add Skill" or "+" button
   - **Expected Result**: Skill form appears

3. **Enter the skill name**
   - Type the name of the skill (e.g., "Python", "Project Management", "Spanish")
   - Use standard, recognizable skill names
   - **Expected Result**: Skill name is entered

4. **Select the skill type**
   - Choose from available categories:
     - **Technical**: Programming languages, software, technical tools
     - **Soft Skills**: Communication, leadership, teamwork
     - **Language**: Spoken/written languages
     - **Certification**: Professional certifications
     - **Domain Knowledge**: Industry-specific knowledge
     - **Tools & Software**: Specific tools and applications
     - **Other**: Skills that don't fit other categories
   - **Expected Result**: Skill type is selected


5. **Select proficiency level**
   - Choose your skill level:
     - **1 - Beginner**: Just starting to learn
     - **2 - Elementary**: Basic understanding and usage
     - **3 - Intermediate**: Comfortable with regular use
     - **4 - Advanced**: High proficiency, can teach others
     - **5 - Expert**: Master level, recognized authority
   - **Expected Result**: Proficiency level is selected

6. **Enter years of experience**
   - Type the number of years you've worked with this skill
   - Can be 0 for newly acquired skills
   - **Expected Result**: Years of experience is entered

7. **Set last used date (optional)**
   - Click the date field
   - Select when you last used this skill
   - Helps show skill currency
   - **Expected Result**: Last used date is selected

8. **Mark as primary skill (optional)**
   - Check "Primary Skill" if this is one of your key skills
   - Primary skills are highlighted on your profile
   - **Expected Result**: Primary skill checkbox is selected

9. **Save the skill**
   - Click "Save" or "Add Skill" button
   - **Expected Result**: Success message appears, skill is added to your profile

### Task 2: Editing an Existing Skill

1. **Locate the skill to edit**
   - Scroll to the Skills section
   - Find the skill you want to modify
   - **Expected Result**: Skill entry is visible

2. **Click the "Edit" button**
   - Look for the edit icon (pencil) or "Edit" button
   - **Expected Result**: Skill form opens with current information

3. **Modify the information**
   - Update any fields: name, type, proficiency, experience, etc.
   - **Expected Result**: Fields are updated with new information

4. **Save your changes**
   - Click "Save" or "Update" button
   - **Expected Result**: Success message appears, changes are saved

### Task 3: Organizing Skills by Type

Skills are automatically organized by type on your profile. Here's how to effectively categorize your skills:

1. **Technical Skills**
   - Programming languages (Python, Java, JavaScript)
   - Technical frameworks (React, Django, .NET)
   - Technical concepts (Machine Learning, Cloud Computing)
   - **Expected Result**: Technical skills are grouped together

2. **Soft Skills**
   - Leadership, Communication, Teamwork
   - Problem Solving, Critical Thinking
   - Time Management, Adaptability
   - **Expected Result**: Soft skills are grouped together

3. **Languages**
   - English, Filipino, Spanish, etc.
   - Specify proficiency level
   - **Expected Result**: Language skills are grouped together

4. **Certifications**
   - Professional certifications (PMP, AWS Certified, etc.)
   - Industry credentials
   - **Expected Result**: Certifications are grouped together

5. **Domain Knowledge**
   - Industry-specific expertise
   - Business domains (Finance, Healthcare, Education)
   - **Expected Result**: Domain knowledge is grouped together

6. **Tools & Software**
   - Specific applications (Microsoft Office, Adobe Creative Suite)
   - Development tools (Git, Docker, Jenkins)
   - **Expected Result**: Tools are grouped together


### Task 4: Deleting a Skill

1. **Locate the skill to delete**
   - Scroll to the Skills section
   - Find the skill you want to remove
   - **Expected Result**: Skill entry is visible

2. **Click the "Delete" button**
   - Look for the delete icon (trash can) or "Delete" button
   - **Expected Result**: Confirmation dialog appears

3. **Confirm deletion**
   - Read the confirmation message
   - Click "Yes" or "Confirm" to proceed
   - Click "No" or "Cancel" to abort
   - **Expected Result**: Skill is deleted if confirmed

4. **Verify deletion**
   - Check that the skill no longer appears in your skills list
   - **Expected Result**: Skill is removed from profile

- **Update Proficiency**: Increase proficiency levels as you improve
- **Mark Primary Skills**: Highlight your strongest 5-10 skills
- **Include Soft Skills**: Don't forget communication, leadership, etc.
- **Add Languages**: Include all languages you speak
- **Track Experience**: Keep years of experience current
- **Update Last Used**: Update when you use skills in projects
- **Remove Outdated Skills**: Delete skills you no longer use or maintain

## Troubleshooting

### Issue: Cannot add skill
**Symptoms**: Add skill button doesn't work or form doesn't appear

**Solution**:
- Ensure you're logged in
- Check internet connection
- Try refreshing the page
- Clear browser cache
- Try a different browser
- Contact support if issue persists

### Issue: Skill not saving
**Symptoms**: Click save but skill doesn't appear in list

**Solution**:
- Check for validation errors (red text)
- Ensure all required fields are filled (name, type, proficiency, years)
- Verify skill name is not empty
- Check that proficiency level is selected
- Try refreshing and re-entering
- Contact support if problem continues

### Issue: Cannot select proficiency level
**Symptoms**: Proficiency dropdown is disabled or not working

**Solution**:
- Ensure skill type is selected first
- Try refreshing the page
- Clear browser cache
- Try a different browser
- Contact support if issue persists

### Issue: Duplicate skills appearing
**Symptoms**: Same skill appears multiple times

**Solution**:
- Delete duplicate entries
- Use exact same name for related skills
- Check skill type - same skill with different types will appear separately
- Contact support if you cannot delete duplicates


### Issue: Skills not appearing in job matches
**Symptoms**: Not receiving job recommendations despite having relevant skills

**Solution**:
- Ensure skills are saved to your profile
- Check that skill names match job requirements
- Verify proficiency level is adequate
- Update years of experience
- Mark relevant skills as primary
- Wait for job matching system to update (may take a few hours)
- Contact support if matches still don't appear

### Issue: Cannot mark skill as primary
**Symptoms**: Primary skill checkbox doesn't work

**Solution**:
- Ensure you're editing an existing skill
- Try unchecking and rechecking
- Refresh the page
- You can have multiple primary skills
- Contact support if issue persists

## Proficiency Level Guidelines

### Level 1 - Beginner
- Just starting to learn the skill
- Can perform basic tasks with guidance
- Limited practical experience
- Still building foundational knowledge

### Level 2 - Elementary
- Basic understanding and usage
- Can perform routine tasks independently
- Some practical experience
- Requires occasional guidance

### Level 3 - Intermediate
- Comfortable with regular use
- Can handle most common scenarios
- Solid practical experience
- Can work independently on standard tasks

### Level 4 - Advanced
- High proficiency in the skill
- Can handle complex scenarios
- Extensive practical experience
- Can teach and mentor others
- Recognized as skilled by peers

### Level 5 - Expert
- Master level proficiency
- Deep understanding of all aspects
- Years of extensive experience
- Recognized authority in the field
- Can innovate and create new approaches
- Sought after for expertise


## How Skills Are Used

### Job Matching
- System matches your skills with job requirements
- Higher proficiency levels increase match scores
- Primary skills are weighted more heavily
- Recent skills (last used date) are prioritized
- You receive notifications for high-match jobs

### Mentorship Pairing
- Mentors are matched with mentees based on skills
- Expert-level skills make you visible as a potential mentor
- Mentees can search for mentors by skill
- Skills help identify expertise areas

### Alumni Directory Search
- Other alumni can search for people with specific skills
- Skills appear on your public profile
- Helps with networking and collaboration
- Enables finding experts in specific areas

### Profile Completeness
- Skills contribute to profile completeness score
- More skills = more complete profile
- Complete profiles rank higher in searches
- Improves visibility in the alumni network

## Additional Notes

### Skill Display
- Skills are grouped by type on your profile
- Sorted by proficiency level and years of experience
- Primary skills are highlighted with badges
- All skills appear on your public profile (if profile is public)

### Data Validation
- Skill name is required (max 100 characters)
- Skill type must be selected
- Proficiency level must be 1-5
- Years of experience must be 0 or positive
- Last used date must be in the past

### Best Practices for Skill Names
- Use standard, industry-recognized names
- Be specific (e.g., "Python" not just "Programming")
- Use proper capitalization
- Avoid abbreviations unless commonly used
- Keep names concise
- Use singular form (e.g., "Skill" not "Skills")

### Recommended Number of Skills
- Minimum: 5-10 skills for basic profile
- Optimal: 15-25 skills for comprehensive profile
- Include mix of technical and soft skills
- Focus on quality over quantity
- Only include skills you can demonstrate

### Updating Skills
- Review skills quarterly
- Add new skills as you learn them
- Update proficiency as you improve
- Remove outdated or unused skills
- Update last used dates regularly
- Adjust primary skills as your focus changes
