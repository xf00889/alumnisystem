# Surveys

## Overview

The Surveys feature allows you to participate in alumni surveys created by administrators. Surveys help gather valuable feedback, track alumni achievements, collect employment data, and understand the alumni community better. You can view available surveys, complete them, and contribute to important research and decision-making processes.

## Who Can Use This Feature

- **User Role**: All authenticated alumni users
- **Permissions Required**: Active user account
- **Prerequisites**: Must be logged in to the system

## Key Features

- View all active surveys
- Complete surveys with various question types
- Track which surveys you've already completed
- Access both internal and external surveys
- Receive confirmation after survey submission

## Available Documentation

- [Taking Surveys](taking-surveys.md) - Learn how to access and complete surveys

## Survey Types

### Internal Surveys
Internal surveys are created and hosted within the alumni system. They feature:
- Multiple question types (text, multiple choice, checkboxes, ratings, dates, etc.)
- Required and optional questions
- Help text for guidance
- Immediate submission confirmation

### External Surveys
External surveys are hosted on third-party platforms (like Google Forms). They feature:
- Direct links to external survey platforms
- Seamless integration with the alumni system
- Tracking of survey completion status

## Question Types

When taking surveys, you may encounter various question types:

1. **Text Answer**: Open-ended text responses
2. **Multiple Choice**: Select one option from a list
3. **Checkboxes**: Select multiple options from a list
4. **Rating Scale**: Rate on a numerical scale (e.g., 1-5)
5. **Likert Scale**: Agreement or satisfaction scales
6. **Date**: Select a specific date
7. **Time**: Select a specific time
8. **File Upload**: Upload documents or files
9. **Email**: Provide email addresses
10. **Number**: Enter numerical values
11. **Phone Number**: Provide phone numbers
12. **Website URL**: Provide web addresses

## Survey Status

Surveys can have different statuses:

- **Active**: Currently available for completion
- **Draft**: Not yet available (admin only)
- **Closed**: No longer accepting responses

## Privacy and Data Usage

- Your survey responses are confidential
- Responses are used for research and improvement purposes
- You can only submit each survey once
- Your IP address may be recorded for security purposes

## Tips for Taking Surveys

- Read all questions carefully before answering
- Pay attention to required questions (marked with *)
- Use the help text provided for guidance
- Take your time - there's no time limit
- Review your answers before submitting
- You cannot edit responses after submission

## Common Use Cases

### Use Case 1: Employment Survey
Administrators may send employment surveys to track alumni career progress. These surveys help understand employment trends and improve career services.

### Use Case 2: Event Feedback
After attending events, you may receive feedback surveys to help improve future events.

### Use Case 3: Alumni Satisfaction
Periodic satisfaction surveys help administrators understand alumni needs and improve services.

### Use Case 4: Research Studies
Academic research surveys help contribute to studies about alumni outcomes and experiences.
