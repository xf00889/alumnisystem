# Browsing Groups

Learn how to find and explore alumni groups using the groups list, search functionality, and filtering options.

## Overview

The groups browsing feature allows you to discover alumni groups based on various criteria such as batch year, course, campus, and group type. You can search for specific groups, filter results, and view group details before deciding to join.

## Who Can Use This Feature

- **Required**: Authenticated alumni account
- **Access Level**: All registered alumni users
- **Prerequisites**: 
  - Verified email address
  - Completed profile setup

## How to Access

1. Log in to your alumni account
2. Click on **"Groups"** in the main navigation menu
3. You will be directed to the Groups List page

**Alternative Access:**
- From your dashboard, click the **"Browse Groups"** button
- Use the search bar and select **"Groups"** from the dropdown

## Groups List Page

### Page Layout

The Groups List page displays:

**Header Section:**
- Total number of groups in the system
- Your group memberships count
- "Create Group" button (if you have permission)

**Filter Panel (Left Sidebar):**
- Search box
- Batch year range selector
- Course filter
- Campus filter
- Group type filter
- Sort options

**Groups Grid (Main Area):**
- Group cards showing:
  - Group name
  - Group type badge
  - Cover image (if available)
  - Member count
  - Brief description
  - "View Details" or "Join" button
  - Your membership status (if applicable)

**Recent Activity Panel (Right Sidebar):**
- Recent group activities
- Your groups quick access
- Suggested groups

## Viewing the Groups List

### Step-by-Step: Browse All Groups

1. Navigate to the Groups page
2. Scroll through the list of available groups
3. View group cards with basic information
4. Click on any group card to view full details

**Expected Result:**
- Groups are displayed in a grid layout
- 12 groups per page by default
- Pagination controls at the bottom

### Group Card Information

Each group card displays:

- **Group Name**: The official name of the group
- **Group Type**: Badge indicating Automatic, Manual, or Hybrid
- **Visibility**: Public, Private, or Restricted indicator
- **Member Count**: Number of active members
- **Description**: Brief overview of the group's purpose
- **Cover Image**: Visual representation of the group
- **Membership Status**: Your current status (if applicable)
  - "Member" - You are an active member
  - "Pending" - Your join request is awaiting approval
  - "Join" button - You are not a member

## Searching for Groups

### Step-by-Step: Search by Keyword

1. Locate the search box at the top of the Groups List page
2. Enter keywords related to:
   - Group name
   - Description content
   - Tags
3. Press **Enter** or click the search icon
4. View filtered results matching your search

**Search Tips:**
- Use specific keywords for better results
- Try different variations of group names
- Search by batch year (e.g., "2020")
- Search by course abbreviation (e.g., "BSIT")
- Use tags if you know them

**Example Searches:**
- "Computer Science 2020" - Find CS batch groups
- "Engineering" - Find all engineering-related groups
- "Main Campus" - Find groups for main campus
- "Basketball" - Find sports-related groups

**Expected Result:**
- Groups matching your search criteria are displayed
- Non-matching groups are hidden
- Search term is highlighted in results
- "Clear search" option appears

## Filtering Groups

### Filter by Batch Year Range

**Step-by-Step:**

1. Locate the **"Batch Year"** filter in the left sidebar
2. Select **"Start Year"** from the dropdown
3. Select **"End Year"** from the dropdown
4. Click **"Apply Filter"**

**Example:**
- Start Year: 2015
- End Year: 2020
- Result: Shows groups for batches 2015-2020

**Expected Result:**
- Only groups within the selected year range are displayed
- Filter badge shows active filter
- "Clear filter" option appears

### Filter by Course

**Step-by-Step:**

1. Locate the **"Course"** filter in the left sidebar
2. Click the dropdown menu
3. Select a course from the list
4. Groups are automatically filtered

**Available Courses:**
- All courses offered at NORSU campuses
- Courses are populated from existing groups

**Expected Result:**
- Only groups for the selected course are displayed
- Other courses are hidden
- Filter can be cleared to show all courses

### Filter by Campus

**Step-by-Step:**

1. Locate the **"Campus"** filter in the left sidebar
2. Click the dropdown menu
3. Select a campus:
   - NORSU Main Campus
   - NORSU Guihulngan Campus
   - NORSU Siaton Campus
   - NORSU Pamplona Campus
   - NORSU Bais Campus
   - NORSU BSC (Bayawan-Sta. Catalina) Campus
4. Groups are automatically filtered

**Expected Result:**
- Only groups for the selected campus are displayed
- Campus-specific groups are highlighted
- Filter can be cleared to show all campuses

### Filter by Group Type

**Step-by-Step:**

1. Locate the **"Group Type"** filter in the left sidebar
2. Click the dropdown menu
3. Select a group type:
   - **Automatic**: Batch-based groups
   - **Manual**: User-created interest groups
   - **Hybrid**: Combined automatic and manual
4. Groups are automatically filtered

**Expected Result:**
- Only groups of the selected type are displayed
- Type badge is visible on each group card
- Filter can be cleared to show all types

### Combining Multiple Filters

You can apply multiple filters simultaneously:

**Step-by-Step:**

1. Select a batch year range
2. Choose a course
3. Select a campus
4. Pick a group type
5. All filters are applied together

**Example Combination:**
- Batch: 2018-2020
- Course: BS Computer Science
- Campus: Main Campus
- Type: Automatic

**Expected Result:**
- Only groups matching ALL criteria are displayed
- Multiple filter badges appear
- "Clear all filters" button is available

## Sorting Groups

### Available Sort Options

**Sort by:**
- **Newest First** (Default): Recently created groups appear first
- **Oldest First**: Longest-standing groups appear first
- **Most Members**: Groups with highest member count first
- **Most Active**: Groups with most recent activity first

### Step-by-Step: Change Sort Order

1. Locate the **"Sort by"** dropdown in the filter panel
2. Click the dropdown menu
3. Select your preferred sort option
4. Groups are automatically reordered

**Expected Result:**
- Groups are reordered based on selected criteria
- Sort selection is remembered during your session
- Pagination resets to page 1

## Viewing Group Details

### Step-by-Step: View Full Group Information

1. Find a group that interests you
2. Click on the group card or **"View Details"** button
3. You are redirected to the Group Detail page

**Group Detail Page Shows:**
- Full group description
- Complete member list
- Group statistics
- Recent activities
- Upcoming events
- Group files
- Join/Leave button
- Group settings (if you're an admin)

**Expected Result:**
- Detailed information about the group is displayed
- You can decide whether to join
- Navigation back to groups list is available

## My Groups Section

### Viewing Your Groups

**Step-by-Step:**

1. On the Groups List page, locate the **"My Groups"** section
2. View groups where you are a member
3. Click on any group to access it directly

**My Groups Shows:**
- Groups where your membership is approved
- Quick access to your active groups
- Membership role indicator (Member, Moderator, Admin)

**Expected Result:**
- Fast access to groups you belong to
- No need to search for your groups
- Direct navigation to group pages

## Pagination

### Navigating Multiple Pages

**Step-by-Step:**

1. Scroll to the bottom of the groups list
2. View pagination controls showing:
   - Current page number
   - Total number of pages
   - Previous/Next buttons
   - Page number links
3. Click **"Next"** to view the next page
4. Click a page number to jump to that page
5. Click **"Previous"** to go back

**Expected Result:**
- 12 groups displayed per page
- Smooth navigation between pages
- Current page is highlighted
- Filters and search persist across pages

## Tips and Best Practices

### Finding Your Batch Group
1. Use the batch year filter to narrow results
2. Select your course from the course filter
3. Choose your campus
4. Look for automatic groups matching your profile

### Discovering Interest Groups
1. Use the search box with interest keywords
2. Filter by "Manual" group type
3. Browse group descriptions for relevant topics
4. Check member counts for active groups

### Exploring New Groups
1. Sort by "Newest First" to see recent groups
2. Check the "Recent Activity" panel for active groups
3. Look for groups with growing member counts
4. Read group descriptions carefully

### Efficient Browsing
1. Use multiple filters to narrow results quickly
2. Save time by checking "My Groups" first
3. Use search for specific group names
4. Clear filters to start fresh searches

## Common Use Cases

### Finding Your Graduation Year Group

**Scenario:** You want to connect with your batchmates

**Steps:**
1. Go to Groups List page
2. Set batch year filter to your graduation year
3. Select your course
4. Choose your campus
5. Look for the automatic group for your batch
6. Click "View Details" to verify it's your group
7. Click "Join" to request membership

### Discovering Professional Groups

**Scenario:** You want to network with alumni in your field

**Steps:**
1. Use the search box with your profession or industry
2. Filter by "Manual" group type
3. Review group descriptions for relevance
4. Check member counts and activity levels
5. Join groups that match your interests

### Exploring Campus-Specific Groups

**Scenario:** You want to connect with alumni from your campus

**Steps:**
1. Select your campus from the campus filter
2. Browse all groups for that campus
3. Look for campus-wide groups or events
4. Join groups that interest you

## Troubleshooting

### No Groups Found

**Problem:** Search or filters return no results

**Solutions:**
- Clear all filters and try again
- Broaden your search terms
- Check if you're using correct batch years
- Try different filter combinations
- Verify groups exist for your criteria

### Cannot See Private Groups

**Problem:** You know a group exists but can't find it

**Solutions:**
- Private groups are only visible to members
- Ask a member to invite you
- Contact the group admin for access
- Check if you have the correct group name

### Search Not Working

**Problem:** Search doesn't return expected results

**Solutions:**
- Check your spelling
- Try different keywords
- Use shorter, more general terms
- Clear search and use filters instead
- Refresh the page and try again

### Filters Not Applying

**Problem:** Filters don't seem to work

**Solutions:**
- Click "Apply Filter" if required
- Refresh the page
- Clear browser cache
- Try one filter at a time
- Check if any groups match your criteria

### Page Loading Slowly

**Problem:** Groups list takes time to load

**Solutions:**
- Check your internet connection
- Reduce number of active filters
- Clear browser cache
- Try a different browser
- Contact support if issue persists
