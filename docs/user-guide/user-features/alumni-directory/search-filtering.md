# Search Filtering

This guide explains how to use the powerful search and filtering features to find specific alumni in the NORSU Alumni Directory.

## Overview

The Alumni Directory provides comprehensive search and filtering capabilities to help you quickly find alumni based on various criteria. You can search by name, filter by academic information, location, employment status, and more.

## Search Functionality

### Using the Search Bar

The search bar allows you to find alumni by entering keywords related to their information.

#### Step 1: Locate the Search Bar

1. Navigate to the Alumni Directory
2. Find the search section at the top of the page
3. Look for the input field labeled **"Search Alumni"**

![Search Bar Location](../../screenshots/alumni-search-bar.png)

#### Step 2: Enter Search Terms

1. Click in the search input field
2. Type your search query
3. You can search for:
   - **First name** (e.g., "Juan")
   - **Last name** (e.g., "Dela Cruz")
   - **Full name** (e.g., "Juan Dela Cruz")
   - **Course name** (e.g., "Computer Science")
   - **College name** (e.g., "Engineering")
   - **City** (e.g., "Dumaguete")
   - **Province** (e.g., "Negros Oriental")
   - **Company name** (e.g., "Google")
   - **Job title** (e.g., "Software Engineer")

**Example Searches**:
- "Maria" - Finds all alumni with "Maria" in their name
- "Computer Science" - Finds all Computer Science graduates
- "Engineering" - Finds all alumni from engineering-related colleges
- "Dumaguete" - Finds all alumni located in Dumaguete

#### Step 3: Execute the Search

1. After entering your search term, click the **"Search"** button (blue button with magnifying glass icon)
2. Or press **Enter** on your keyboard
3. The page will reload with filtered results

**Expected Result**: The directory displays only alumni matching your search criteria.

![Search Results](../../screenshots/alumni-search-results.png)

### Search Behavior

- **Case-Insensitive**: Searches work regardless of capitalization
- **Partial Matching**: Finds results containing your search term anywhere in the field
- **Multiple Fields**: Searches across name, course, college, location, and employment fields simultaneously
- **Real-Time**: Results update immediately after clicking search

### Search Tips

- **Be Specific**: Use specific terms for better results (e.g., "BS Computer Science" instead of just "Computer")
- **Try Variations**: If you don't find results, try different spellings or abbreviations
- **Use Partial Names**: You don't need to type the complete name
- **Combine with Filters**: Use search together with filters for more precise results

## Filtering Options

### Available Filters

The directory provides five main filter categories:

1. **Graduation Year** - Filter by year of graduation
2. **Course** - Filter by degree program
3. **College** - Filter by college/faculty
4. **Campus** - Filter by NORSU campus
5. **Province** - Filter by current location
6. **Employment Status** - Filter by current employment situation

### Filter Location

All filters are located in the search section at the top of the directory page, arranged in a grid layout for easy access.

## Filtering by Graduation Year

### How to Use

1. Locate the **"Graduation Year"** dropdown
2. Click on the dropdown to see available years
3. Select a specific year from the list
4. Click **"Search"** to apply the filter

**Available Options**:
- "All Years" (default - shows all alumni)
- Individual years (e.g., 2020, 2019, 2018, etc.)
- Years are listed in descending order (most recent first)

**Example Use Cases**:
- Find classmates from your graduation year
- Connect with recent graduates
- Find alumni from a specific era

![Graduation Year Filter](../../screenshots/alumni-filter-year.png)

## Combining Multiple Filters

### How to Combine Filters

You can use multiple filters simultaneously for more precise results:

1. Select your first filter (e.g., Graduation Year: 2020)
2. Select additional filters (e.g., College: Engineering, Province: Negros Oriental)
3. Optionally add a search term
4. Click **"Search"** to apply all filters

**Example Combinations**:
- **Find Classmates**: Same graduation year + same course
- **Local Networking**: Same college + same province
- **Career Connections**: Same course + employed full-time
- **Recent Graduates in Your Field**: Recent year + your course + your location

![Multiple Filters Applied](../../screenshots/alumni-multiple-filters.png)

### Filter Logic

- **AND Logic**: All selected filters must match (results meet ALL criteria)
- **Cumulative**: Each additional filter narrows down the results
- **Search + Filters**: Search term is applied along with all selected filters

## Active Filters Display

### Viewing Active Filters

When you have filters applied:
- An **"Active Filters"** section appears below the search form
- Each active filter is displayed as a colored tag
- Tags show the filter type and selected value

**Example Tags**:
- "Year: 2020"
- "College: CAS"
- "Course: BS Computer Science"
- "Province: Negros Oriental"

![Active Filters Display](../../screenshots/alumni-active-filters.png)

### Removing Individual Filters

To remove a specific filter without clearing all:

1. Locate the filter tag in the "Active Filters" section
2. Click the **"X"** button on the tag
3. The filter is removed and results update automatically

**Expected Result**: The directory refreshes with that filter removed, but other filters remain active.

## Clearing All Filters

### Using the Clear Button

To remove all filters and return to the full directory:

1. Locate the **"Clear"** button (red button with X icon)
2. Click the button
3. All filters and search terms are removed
4. The directory displays all alumni again

**Note**: The Clear button only appears when at least one filter or search term is active.

![Clear Filters Button](../../screenshots/alumni-clear-button.png)

## Filter Results Statistics

### Understanding the Statistics

The statistics dashboard updates based on your applied filters:

- **Total Alumni**: Shows count matching your filters
- **Graduation Years**: Shows number of years represented in filtered results
- **Courses**: Shows number of courses in filtered results
- **Provinces**: Shows number of provinces in filtered results

**Example**:
- Before filters: "Total Alumni: 1,500"
- After filtering by Engineering: "Total Alumni: 250"

This helps you understand how many results match your criteria.

## Advanced Filtering Techniques

### Finding Specific Alumni

**Scenario**: Find a classmate named "Maria" who graduated in 2018 from Computer Science

1. Enter "Maria" in the search bar
2. Select "2018" from Graduation Year
3. Select "BS Computer Science" from Course

**Scenario**: Find employed alumni in your city for job networking

1. Select your province from the Province filter
2. Select "Employed Full-Time" from Employment Status

4. Click Search

## Filter Persistence

### How Filters Persist

- **Pagination**: Filters remain active when navigating between pages
- **Profile Viewing**: Filters are preserved when you return from viewing a profile
- **Browser Back**: Filters are maintained when using browser back button
- **Session**: Filters reset when you navigate away from the directory or refresh

### URL Parameters

Filters are stored in the URL, allowing you to:
- **Bookmark**: Save specific filter combinations
- **Share**: Send filtered directory links to others
- **Return**: Come back to the same filtered view

- **Start Broad**: Begin with fewer filters and narrow down
- **Check Statistics**: Use the stats dashboard to gauge result size

## Troubleshooting

### No Results Found

If your filters return no results:
- **Remove Filters**: Try removing one filter at a time
- **Broaden Search**: Use less specific search terms
- **Check Spelling**: Verify search terms are spelled correctly
- **Try Alternatives**: Use different filter combinations

### Filters Not Working

If filters don't seem to apply:
- **Click Search**: Ensure you clicked the Search button
- **Check Selection**: Verify you selected an option (not just opened the dropdown)
- **Refresh Page**: Try refreshing the browser
- **Clear Cache**: Clear browser cache if issues persist

### Unexpected Results

If results don't match your expectations:
- **Review Active Filters**: Check the Active Filters section
- **Verify Logic**: Remember filters use AND logic (all must match)
- **Check Search Term**: Ensure search term isn't too restrictive
- **Clear and Retry**: Clear all filters and start over
