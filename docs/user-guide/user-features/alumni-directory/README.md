# Alumni Directory

The Alumni Directory is a comprehensive feature that allows you to search, browse, and connect with fellow NORSU graduates. This powerful tool helps you build your professional network, find mentors, and stay connected with the alumni community.

## Overview

The Alumni Directory provides:
- **Search and Filter**: Find alumni by name, course, graduation year, college, location, and more
- **Profile Viewing**: View detailed profiles of fellow alumni including their education, work experience, and achievements
- **Connection Features**: Send connection requests and build your professional network
- **Mentor Discovery**: Find verified mentors in your field of interest

## Who Can Use This Feature

- **Access**: Authenticated alumni users
- **Prerequisites**: You must be logged in with a verified alumni account

## Key Features

### 1. Comprehensive Search
Search alumni by multiple criteria including name, course, college, location, and employment status.

### 2. Advanced Filtering
Filter results by:
- Graduation year
- Course/Program
- College
- Campus
- Province/Location
- Employment status

### 3. Profile Viewing
View detailed alumni profiles including:
- Personal and contact information
- Education history
- Work experience and career path
- Skills and achievements
- Uploaded documents

### 4. Networking Tools
- Send connection requests
- Message connected alumni
- Find mentors in your field

### 5. Statistics Dashboard
View directory statistics including:
- Total alumni count
- Number of graduation years represented
- Number of courses offered
- Geographic distribution

## Getting Started

1. **Access the Directory**: Navigate to the Alumni Directory from the main menu
2. **Browse Alumni**: View the list of alumni displayed in card format
3. **Use Filters**: Apply filters to narrow down your search
4. **View Profiles**: Click on any alumni card to view their full profile
5. **Connect**: Send connection requests to alumni you'd like to network with

## Tips and Best Practices

- **Use Multiple Filters**: Combine filters to find specific alumni (e.g., same course and graduation year)
- **Update Your Profile**: Keep your own profile updated so others can find and connect with you
- **Respect Privacy**: Only use contact information for professional networking purposes
- **Build Your Network**: Regularly connect with alumni in your field or area of interest
- **Engage with Mentors**: Look for verified mentors who can provide career guidance
