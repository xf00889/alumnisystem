# Browsing

This guide explains how to access and navigate the Alumni Directory to find and view fellow NORSU graduates.

## Overview

The Alumni Directory displays all registered alumni in an easy-to-browse card layout. Each card shows key information about an alumnus, making it simple to find people you want to connect with.

## Accessing the Directory

### Step 1: Navigate to Alumni Directory

1. Log in to your NORSU Alumni account
2. From the main navigation menu, click on **"Alumni Directory"** or **"Directory"**
3. The directory page will load, displaying all alumni

**Expected Result**: You'll see the Alumni Directory page with:
- A header showing "NORSU Alumni Directory"
- Search and filter options at the top
- Statistics cards showing directory metrics
- A grid of alumni cards below

![Alumni Directory Main Page](../../screenshots/alumni-directory-main.png)

## Understanding the Directory Layout

### Header Section

The header displays:
- **Title**: "NORSU Alumni Directory"
- **Subtitle**: "Find and connect with fellow NORSU graduates"

### Statistics Dashboard

Four statistics cards show:
- **Total Alumni**: Total number of registered alumni
- **Graduation Years**: Number of different graduation years represented
- **Courses**: Number of different courses/programs
- **Provinces**: Number of provinces where alumni are located

These statistics update based on your applied filters.

### Alumni Cards

Each alumni card displays:
- **Profile Photo**: Alumni's profile picture (or default avatar)
- **Full Name**: First and last name
- **Username**: System username
- **College**: The college they graduated from (e.g., "College of Engineering and Architecture")
- **Course**: Their degree program
- **Graduation Year**: Year they graduated (e.g., "Class of 2020")
- **Location**: City, province, and country (if provided)
- **Job Title**: Current position (if provided)
- **Mentor Badge**: Green badge indicating verified mentors
- **Action Buttons**: 
  - "View Profile" - Opens the full profile
  - "Connect" or "Message" - Networking options

![Alumni Card Example](../../screenshots/alumni-card.png)

## Viewing Alumni List

### Default View

When you first access the directory:
- All alumni are displayed in a grid layout
- Cards are arranged in rows (typically 3-4 cards per row on desktop)
- The most recent registrations may appear first
- 12 alumni are shown per page by default

### Card Grid Layout

The alumni cards are displayed in a responsive grid:
- **Desktop**: 3-4 cards per row
- **Tablet**: 2 cards per row
- **Mobile**: 1 card per row

### Scrolling and Viewing

- Scroll down to view more alumni cards
- The layout automatically adjusts to your screen size
- Cards maintain consistent sizing and spacing

## Pagination

### Understanding Pagination

When there are more than 12 alumni:
- Pagination controls appear at the bottom of the page
- Page numbers show how many pages of results exist
- Current page is highlighted

### Navigation Controls

The pagination section includes:
- **First Page (<<)**: Jump to the first page
- **Previous (<)**: Go to the previous page
- **Page Numbers**: Click any number to jump to that page
- **Next (>)**: Go to the next page
- **Last Page (>>)**: Jump to the last page

![Pagination Controls](../../screenshots/alumni-pagination.png)

### Using Pagination

1. **View Next Page**:
   - Click the ">" button or the next page number
   - The page will reload with the next set of alumni

2. **Jump to Specific Page**:
   - Click on any visible page number
   - The page will load that specific set of results

3. **Return to First Page**:
   - Click the "<<" button
   - You'll return to page 1

**Note**: Pagination preserves your search and filter settings. When you navigate between pages, your filters remain active.

## Action Buttons

### View Profile Button
- **Color**: Blue (primary color)
- **Icon**: User icon
- **Action**: Opens the full alumni profile page
- **Available**: For all alumni

### Connect Button
- **Color**: Teal (info color)
- **Icon**: User-plus icon
- **Action**: Sends a connection request
- **Available**: When you're not connected with the alumni
- **States**:
  - "Connect" - Available to send request
  - "Pending" - Request already sent (disabled)
  - "Message" - Already connected (opens messaging)

## Empty State

If no alumni match your search or filters:
- A message displays: "No Alumni Found"
- Subtitle: "There are no alumni records matching your search criteria"
- An icon indicates the empty state
- Clear your filters to see all alumni again

## Tips for Browsing

### Efficient Browsing
- **Scan Cards Quickly**: The card layout makes it easy to scan multiple alumni at once
- **Use Visual Cues**: Look for mentor badges or specific colleges
- **Check Statistics**: Use the stats dashboard to understand the directory scope

### Finding Specific Alumni
- Use the search bar for name-based searches
- Apply filters to narrow down results (see [Search and Filtering](search-filtering.md))
- Combine multiple filters for precise results

### Mobile Browsing
- Cards stack vertically on mobile devices
- All information remains accessible
- Tap cards to view full profiles
- Swipe to scroll through the list

## Keyboard Navigation

For accessibility:
- **Tab**: Move between interactive elements
- **Enter**: Activate buttons and links
- **Arrow Keys**: Navigate pagination (when focused)

## Performance Notes

- The directory loads 12 alumni per page for optimal performance
- Images are optimized for fast loading
- Pagination prevents long page load times
- Filters are applied server-side for efficiency

## Troubleshooting

### Directory Not Loading
- Check your internet connection
- Refresh the page
- Clear your browser cache
- Try logging out and back in

### Cards Not Displaying Properly
- Ensure JavaScript is enabled in your browser
- Try a different browser
- Check if you have any ad blockers that might interfere

### Pagination Not Working
- Ensure you're clicking the pagination links, not browser back/forward
- Check if filters are applied that might limit results
- Refresh the page if pagination seems stuck
