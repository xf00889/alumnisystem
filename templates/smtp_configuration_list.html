About Email Configuration

Email configuration allows the system to send emails for notifications, password resets, and other automated communications. You can choose between SMTP or Brevo API.

SMTP Configuration:
  • Secure Connection: Support for TLS and SSL encryption
  • Multiple Providers: Works with Gmail, Yahoo, Outlook, and custom SMTP servers
  • Test Functionality: Verify configuration before activation
  • Active Management: Only one configuration can be active at a time
Brevo API Configuration:
  • High Deliverability: Professional email service with better delivery rates
  • API-Based: No SMTP server configuration needed
  • Rich Analytics: Track email performance and delivery statistics
  • Easy Setup: Just requires API key and verified sender email
  • ⚠️ Activation Required: Contact Brevo support to activate SMTP API
Security Notes:
  • SMTP: For Gmail/Yahoo: Enable 2FA and use App Passwords. For Outlook: Use your regular password or App Password. Always use TLS (port 587) or SSL (port 465) for security.
  • Brevo: Get your API key from Brevo dashboard → Settings → SMTP & API → API Keys. Make sure your sender email is verified in Brevo.
