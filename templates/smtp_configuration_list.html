About SMTP Configuration

SMTP (Simple Mail Transfer Protocol) configuration allows the system to send emails for notifications, password resets, and other automated communications.

Key Features:
  • Secure Connection: Support for TLS and SSL encryption
  • Multiple Providers: Works with Gmail, Yahoo, Outlook, and custom SMTP servers
  • Test Functionality: Verify configuration before activation
  • Active Management: Only one configuration can be active at a time
  • Reliable Delivery: Ensures emails reach recipients successfully
Security Note:
  • For Gmail/Yahoo: Enable 2FA and use App Passwords
  • For Outlook: Use your regular password or App Password
  • Always use TLS (port 587) or SSL (port 465) for security
