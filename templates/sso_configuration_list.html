Setup Instructions
Google OAuth Setup:
  1. Go to Google Cloud Console
  2. Create a new project or select an existing one
  3. Enable the Google+ API
  4. Go to Credentials → Create Credentials → OAuth 2.0 Client ID
  5. Add authorized redirect URIs: {{ request.scheme }}://{{ request.get_host }}/accounts/google/login/callback/
  6. Copy the Client ID and Client Secret and add them here
Facebook OAuth Setup:
  1. Go to Facebook Developers
  2. Create a new app or select an existing one
  3. Add Facebook Login product
  4. In Settings → Basic, copy the App ID and App Secret
  5. In Facebook Login → Settings, add OAuth redirect URI: {{ request.scheme }}://{{ request.get_host }}/accounts/facebook/login/callback/
  6. Copy the App ID and App Secret and add them here
