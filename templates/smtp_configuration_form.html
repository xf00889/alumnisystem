SMTP Configuration Guide

How to configure SMTP:
  1. Gmail: Use smtp.gmail.com, port 587, enable 2FA and use App Password
  2. Yahoo: Use smtp.mail.yahoo.com, port 587, enable 2FA and use App Password
  3. Outlook: Use smtp-mail.outlook.com, port 587, use your regular password
  4. Custom SMTP: Contact your email provider for server details
Security Notes:
  • TLS (Port 587): Recommended for most providers
  • SSL (Port 465): Alternative encryption method
  • App Passwords: Required for Gmail/Yahoo with 2FA enabled
  • Testing: Use the test button to verify your configuration
